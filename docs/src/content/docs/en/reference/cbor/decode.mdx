## Synchronous Decoding

The `decode` function decodes a CBOR-formatted `Uint8Array` into a JavaScript value.

### Import

```ts
import { decode } from "@tai-kun/surrealdb/cbor";
```

### Syntax

```ts
function decode(input: Uint8Array, options?: DecodeOptions): unknown;
```

#### Parameters

##### `input`

The CBOR-formatted `Uint8Array` to decode into a JavaScript value.

##### `options`

Decoding options.

`mapType?: "Object" | "Map"`
: This option lets you choose whether to decode key-value objects as plain JavaScript objects or `Map` instances. Selecting `"Object"` will decode them into plain objects, while `"Map"` will use `Map` class instances. The default is `"Object"`. While plain object keys can only be strings or numbers (i.e., `string | number`), decoding with this option set to `"Map"` allows keys to be any decoded value.

`reviver?: Reviver | ReviverObject | (Reviver | ReviverObject)[]`
: This option transforms CBOR simple data items or tagged data items into JavaScript values. Simple data items are limited to values from 0 to 19 and 32 to 255. Tagged data items have tags that are either `number` or `bigint` from 0 to 2^64-1. If the function returns the `CONTINUE` symbol obtained with `Symbol.for("@tai-kun/surrealdb/cbor/continue")`, the subsequent functions are called. If nothing can be converted, a `SurrealTypeError` is thrown.

`isSafeMapKey?: (key: unknown, map: Map<unknown, unknown>) => boolean`
: This option provides a function to validate whether a key for a `Map` object is safe. The function takes two arguments: `key` and `map`. The `key` is the value of the key, and `map` is the `Map` object being validated. If this function returns `false`, a `CborUnsafeMapKeyError` is thrown. By default, it considers `"__proto__"` or `"constructor"` as unsafe keys. Note that specifying this option will completely override the default validation. If you want to add your own validation to the default validation, you should also determine if `"__proto__"` or `"constructor"` are unsafe keys.

`isSafeObjectKey?: (key: unknown, obj: Record<string | number, unknown>) => boolean`
: This option provides a function to validate whether a key for a plain object is safe. The function takes two arguments: `key` and `obj`. The `key` is the value of the key, and `obj` is the plain object being validated. If this function returns `false`, a `CborUnsafeObjectKeyError` is thrown. By default, it considers `"__proto__"` or `"constructor"` as unsafe keys. Note that specifying this option will completely override the default validation. If you want to add your own validation to the default validation, you should also determine if `"__proto__"` or `"constructor"` are unsafe keys.

`maxDepth?: number`
: Specifies the maximum depth of the object. The depth is incremented by 1 each time an object or array is entered. The default value is `64`. Although there is no upper limit to the depth because recursion is not used for decoding, it is recommended not to make it too large.

#### Return Value

The JavaScript value decoded from the CBOR-formatted `Uint8Array`.

### Example

#### Decoding with a `Map` object

```ts
import { decode } from "@tai-kun/surrealdb/cbor";

const cbor = new Uint8Array([
  0xa1, // mt: 5, ai: 1
  // key
  0x81, // mt: 4, ai: 1
  0x00, // mt: 0, ai: 0
  // value
  0x00, // mt: 0, ai: 0
]);

const value = decode(bytes, { mapType: "Map" });

console.log(value); // Map(1) { [ 0 ] => 0 }
```

## Asynchronous Decoding

The `decodeStream` function decodes a CBOR-formatted `Uint8Array` into a JavaScript value.

### Import

```ts
import { decodeStream } from "@tai-kun/surrealdb/cbor";
```

### Syntax

```ts
function decodeStream(
  input: ReadableStream<Uint8Array>,
  options?: DecodeStreamOptions,
): unknown;
```

#### Parameters

##### `input`

A readable stream of `Uint8Array`. The Fetch API may provide this stream in the `body` property of the `Response` object in response to a request.

##### `options`

In addition to the synchronous decoding options, the following option can be specified:

`signal?: AbortSignal`
: An AbortSignal to abort the stream.

#### Return Value

A `StatefulPromise` that resolves with the JavaScript value.

### Example

#### Fetch API

```ts
import { decodeStream } from "@tai-kun/surrealdb/cbor";

const response = await fetch("http://localhost:8000/rpc", {
  method: "POST",
  headers: {
    Accept: "application/cbor",
    // ... more headers
  },
  // ... more properties
})

const result = await decodeStream(resp.body);
```
