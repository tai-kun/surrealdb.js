---
title: isBrowser
---

The `isBrowser` function determines whether the currently running JavaScript runtime is a browser environment.

### Import

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function isBrowser(): boolean;
```

#### Arguments

None.

#### Return Value

Returns `true` if the environment is a browser, `false` otherwise.

#### Example

The following example checks whether the environment is a browser:

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";

if (isBrowser()) {
  document;
}
```

### Explanation

The function determines a browser environment by checking if the global variable `document` is not `undefined`. This approach is more reliable than checking for the existence of the `window` object, as some server runtimes, like Deno, add a global `window` object. The Remix web framework refers to this technique as a [Document Guard](https://remix.run/docs/en/main/guides/constraints#document-guard).
