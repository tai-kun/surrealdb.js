`Simple` represents a simple value identified by an integer from 0 to 255.

### Import

```ts
import { Simple } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

Creates a new `Simple` object.

```ts
new Simple(value: number);
```

#### Arguments

##### `value`

An integer value between 0 and 255.

#### Return Value

When called with `new`, `Simple` returns its instance.

### `.value` <Badge variant="instance" /><Badge variant="property" />

An integer value between 0 and 255 identifying the `Simple` object.
