`Tagged` represents a custom value identified by an integer between 0 and 2^64-1.

### Import

```ts
import { Tagged } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

```ts
new Tagged<T>(tag: number | BigInt, value: T);
```

#### Arguments

##### `tag`

An integer between 0 and 2^64-1 that identifies the tagged data item. Note that in JavaScript, use BigInt to represent numbers greater than 2^53-1.

##### `value`

The value of the tagged data item.

#### Returns

When called through `new`, `Tagged` returns its instance.

### `.tag` <Badge variant="instance" /><Badge variant="property" />

`.tag` is a property that holds the integer between 0 and 2^64-1 that identifies the tagged data item.

### `.value` <Badge variant="instance" /><Badge variant="property" />

`.value` is a property that stores the value of the tagged data item.

### `.toCBOR()` <Badge variant="instance" /><Badge variant="method" />

`.toCBOR` is a function that returns information for encoding to CBOR.

```ts
toCBOR(): [tag: number | bigint, value: unknown];
```

#### Arguments

None.

#### Returns

Returns information for encoding to CBOR.
