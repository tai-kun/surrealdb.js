## quoteStr

`quoteStr` is a function that encloses a string in quotation marks. For convenience, it also encloses an empty string in quotation marks.

### Import

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function quoteStr(str: string): string;
```

This function encloses a string in quotation marks (`"` or `'`).

#### Arguments

##### `str`

The string to enclose in quotation marks.

#### Return Value

Returns the string enclosed in quotation marks. The quotation marks used for endisconnecting depend on whether the given string contains a single quote (`'`):

- If the given string is empty:
  - Returns `''`.
- If the given string does **not** contain a single quote:
  - Encloses in single quotes.
- If the given string **contains** a single quote:
  - Encloses in double quotes.
- If the given string contains both single and double quotes:
  - Encloses in double quotes.
  - Double quotes are escaped with a backslash (`\`).

#### Example

The following example encloses a string that does **not** contain a single quote in single quotes:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(``));
console.log(quoteStr(`a`));
console.log(quoteStr(`ğ©¸½`));
console.log(quoteStr(`ğŸ‘ğŸ½`));
console.log(quoteStr(`ãƒã‚š`));
```

Output:

```js
''
'a'
'ğ©¸½'
'ğŸ‘ğŸ½'
'ãƒã‚š'
```

The following example encloses a string that **contains** a single quote in double quotes:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's`));
console.log(quoteStr(`ğ©¸½'s`));
console.log(quoteStr(`'ğŸ‘ğŸ½`));
console.log(quoteStr(`'ãƒã‚š`));
```

Output:

```js
"cat's"
"ğ©¸½'s"
"'ğŸ‘ğŸ½"
"'ãƒã‚š"
```

The following example encloses a string that contains both single and double quotes in double quotes, escaping them as necessary:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's "toy"`));
console.log(quoteStr(`'\\"\\`));
console.log(quoteStr(`ğ©¸½'s "feed"`));
console.log(quoteStr(`'ğŸ‘ğŸ½"`));
console.log(quoteStr(`'ãƒã‚š"`));
```

Output:

```js
"cat's \"toy\""
"'\\\"\\"
"ğ©¸½'s \"feed\""
"'ğŸ‘ğŸ½\""
"'ãƒã‚š\""
```

## escapeKey

`escapeKey` is similar to [`quoteStr`](/en/reference/utils/quote-str/) but is used for object keys. For convenience, it also encloses an empty string in quotation marks.

### Import

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeKey(key: string): string;
```

#### Arguments

##### `key`

The object key.

#### Return Value

Returns the escaped string. Encloses it in double quotes as necessary:

- If the given string is empty:
  - Returns `""`.
- If the given string consists only of alphanumeric characters and underscores:
  - Returns the string as is.
- If the given string contains characters other than alphanumeric characters and underscores:
  - Encloses in double quotes.
- If the given string contains a backslash (`\`):
  - Escapes it.
- If the given string contains a double quote:
  - Escapes it.

#### Example

The following example gets the string as is because it consists only of alphanumeric characters and underscores:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(`123`));
console.log(escapeKey(`foo_bar`));
```

Output:

```js
123
foo_bar
```

The following example encloses the string in double quotes, escaping it as necessary:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(``));
console.log(escapeKey(`foo-bar`));
console.log(escapeKey(`'\\"\\`));
console.log(escapeKey(`ğ©¸½'s "feed"`));
console.log(escapeKey(`'ğŸ‘ğŸ½"`));
console.log(escapeKey(`'ãƒã‚š"`));
```

Output:

```js
""
"foo-bar"
"'\\"\"
"ğ©¸½'s \"feed\""
"'ğŸ‘ğŸ½\""
"'ãƒã‚š\""
```

## escapeIdent

`escapeIdent` is similar to [`quoteStr`](/en/reference/utils/quote-str/) but is used for names of namespaces, databases, tables, and fields. For convenience, it also encloses an empty string in backticks.

### Import

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeIdent(ident: string): string;
```

#### Arguments

##### `ident`

The identifier.

#### Return Value

Returns the escaped string. Encloses it in backticks (<code>`</code>) as necessary:

- If the given string is empty:
  - Returns <code>``</code>.{/* TODO(tai-kun): Fix the issue where this becomes " */}
- If the given string does not start with a digit and consists only of alphanumeric characters and underscores:
  - Returns the string as is.
- If the given string contains a backtick (<code>`</code>):
  - Escapes it.
- If the given string starts with a digit or contains characters other than alphanumeric characters and underscores:
  - Encloses in backticks (<code>`</code>).

#### Example

The following example gets the string as is because it does not start with a digit and consists only of alphanumeric characters and underscores:

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";

```

Output:


The following example encloses the string in backticks (<code>`</code>), escaping it as necessary:

