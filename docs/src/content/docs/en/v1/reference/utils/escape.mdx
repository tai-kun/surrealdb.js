## quoteStr

The `quoteStr` function quotes a string. For convenience, it also quotes empty strings.

### Import

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function quoteStr(str: string): string;
```

Quotes a string with either `"` or `'`.

#### Arguments

##### `str`

The string to quote.

#### Return Value

Returns the quoted string. The quote character depends on whether the given string contains single quotes (`'`):

- When the given string is empty
  - Returns `''`.
- When the given string does **not** contain single quotes
  - Quotes with single quotes.
- When the given string does **contain** single quotes
  - Quotes with double quotes.
- When the given string contains both single and double quotes
  - Quotes with double quotes.
  - Escapes double quotes with backslashes (`\`).

#### Example

The following example shows how to quote a string that **does not** contain single quotes:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(``));
console.log(quoteStr(`a`));
console.log(quoteStr(`ğ©¸½`));
console.log(quoteStr(`ğŸ‘ğŸ½`));
console.log(quoteStr(`ãƒã‚š`));
```

Output:

```js
''
'a'
'ğ©¸½'
'ğŸ‘ğŸ½'
'ãƒã‚š'
```

The following example shows how to quote a string that **does** contain single quotes:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's`));
console.log(quoteStr(`ğ©¸½'s`));
console.log(quoteStr(`'ğŸ‘ğŸ½`));
console.log(quoteStr(`'ãƒã‚š`));
```

Output:

```js

"ğ©¸½'s"
"'ğŸ‘ğŸ½"
"'ãƒã‚š"
```

The following example shows how to quote a string that contains both single and double quotes, escaping when necessary:

```ts

console.log(quoteStr(`cat's "toy"`));
console.log(quoteStr(`'\\"\\`));
console.log(quoteStr(`ğ©¸½'s "feed"`));
console.log(quoteStr(`'ğŸ‘ğŸ½"`));
console.log(quoteStr(`'ãƒã‚š"`));
```

```js
"'ğŸ‘ğŸ½\""
```

## escapeKey

The `escapeKey` function is similar to [`quoteStr`](/en/reference/utils/quote-str/), but is used for object keys. For convenience, it also quotes empty strings.

### Import

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeKey(key: string): string;
```

#### Arguments

##### `key`

The object key.

#### Return Value

Returns the escaped string, quoting with double quotes if necessary:

- When the given string is empty
  - Returns `""`.
- When the given string consists only of alphanumeric characters and underscores
  - Returns the string as is.
- When the given string contains anything other than alphanumeric characters and underscores
  - Quotes with double quotes.
- When the given string contains backslashes (`\`)
  - Escapes them.
- When the given string contains double quotes
  - Escapes them.

#### Example

The following example shows how to escape a string that consists only of alphanumeric characters and underscores, resulting in the string as is:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(`123`));
console.log(escapeKey(`foo_bar`));
```

Output:

```js
123
foo_bar
```

The following example shows how to escape a string, quoting with double quotes and escaping when necessary:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(``));
console.log(escapeKey(`foo-bar`));
console.log(escapeKey(`'\\"\\`));
console.log(escapeKey(`ğ©¸½'s "feed"`));
console.log(escapeKey(`'ğŸ‘ğŸ½"`));
console.log(escapeKey(`'ãƒã‚š"`));
```

Output:

```js
""
"foo-bar"
"'\\\"\\"
"ğ©¸½'s \"feed\""
"'ğŸ‘ğŸ½\""
"'ãƒã‚š\""
```

| ASCII | a | 1 | `[ 0x61 ]` |
| Surrogate Pair | ğ©¸½ | 2 | `[ 0xF0, 0xA9, 0xB8, 0xBD ]` |
| Variation Selector | ğŸ‘ğŸ½ | 4 | `[ 0xF0, 0x9F, 0x91, 0x8D, 0xF0, 0x9F, 0x8F, 0xBD ]` |
