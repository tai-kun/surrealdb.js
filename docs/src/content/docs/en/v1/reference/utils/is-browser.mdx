The `isBrowser` function checks whether the current JavaScript runtime environment is a browser environment.

### Import

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function isBrowser(): boolean;
```

#### Arguments

None.

#### Return Value

Returns `true` if the current environment is a browser, otherwise `false`.

#### Example

The following example checks if the current environment is a browser:

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";

if (isBrowser()) {
  document;
}
```

### Explanation

The function determines whether the current environment is a browser by checking if the global variable `document` is defined. For example, Deno adds the `window` object globally, so checking for the presence of the `window` object would incorrectly identify a server-side runtime as a browser. Remix, a web framework, calls this [Document Guard](https://remix.run/docs/en/main/guides/constraints#document-guard).
