`isSafeNumber` checks if a given number is a safe number for arithmetic operations.

### Import

```ts
import { isSafeNumber } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function isSafeNumber(value: number): boolean;
```

#### Arguments

##### `value`

The number to be validated.

#### Return Value

Returns `false` if `value` is:

- `NaN`
- Infinite
