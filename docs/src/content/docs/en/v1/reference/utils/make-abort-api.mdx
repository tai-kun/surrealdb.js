`makeAbortApi` is a function that creates an abort signal and an associated abort function.

### Import

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function makeAbortApi(signal?: AbortSignal): [
  signal: AbortSignal,
  abort: (reason?: unknown) => void,
];
```

#### Arguments

##### `signal`

The abort signal to associate with.

#### Return Value

Returns an array with the following order:

##### `signal`

The abort signal. It is not aborted by default.

##### `abort`

Dispatches the `"abort"` event to the abort signal. It optionally accepts one argument, `reason`, which can be used to convey the reason for the abort via the `"abort"` event.

#### Example

The following example creates an Abort API without arguments:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const [signal, abort] = makeAbortApi();
abort();
```

The above example is almost the same as `new AbortController()`. The difference from `AbortController` is that `makeAbortApi` returns an array instead of an object.

The following example creates an Abort API with an associated abort signal:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

controller.abort("test");
```

Output:

```js
controller.signal test
signal test
```

The abort event propagates from the Abort API to the associated abort signal when aborting from the Abort API:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal, abort] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

abort("test");
```

Output:

```js
controller.signal test
signal test
```
