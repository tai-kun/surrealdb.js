`toSurql` is a function that converts JavaScript values to SurrealQL strings.

### Import

```ts
import { toSurql } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function toSurql(value: unknown): string;
```

#### Arguments

##### `value`

A JavaScript value. It accepts one of the following values:

- `string`
- `number`
- `bigint`
- `boolean`
- `undefined`
- `null`
- `{ toSurql(): string }`
- `Date`
- `{ toJSON(): string }`
- `unknown[]`
- `{ [key: string]: unknown }`
- `Set<unknown>`
- `Map<unknown, unknown>`

`unknown` represents a value that can be any of the types listed above.

The order of the types listed above indicates the priority of conversion. This means that if a plain object has both `.toJSON` and `.toSurql` functions, the `.toSurql` function will be used.

The [`is-plain-obj`](https://github.com/sindresorhus/is-plain-obj) package is used to identify plain objects. Therefore, if a plain object does not have `.toSurql` or `.toJSON` functions and has `Symbol.toStringTag` or `Symbol.iterator`, it will not be treated as an object and an error will be thrown.

#### Return value

A SurrealQL string.

#### Example

```ts
import { toSurql } from "@tai-kun/surrealdb/utils";

const surql = toSurql({
  bigint: 9007199254740992n, // Number.MAX_SAFE_INTEGER + 1
  boolean: [true, false],
  date: new Date("2024-06-01T12:34:56.789Z"),
  map: new Map([
    [0, {
      toJSON: () => [0, 1],
    }],
  ]),
  null: null,
  number: 123,
  set: new Set([{
    toSurql: () => `<bytes>"hello"`,
  }]),
  string: "文字列",
  undefined: undefined,
});

console.log(surql);
```

Output:

```text
{bigint:9007199254740992,boolean:[true,false],date:d'2024-06-01T12:34:56.789Z',map:{0:[0,1]},null:NULL,number:123,set:[<bytes>"hello"],string:'文字列',undefined:NONE}
```
