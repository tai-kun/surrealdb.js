## Redirects

The following rules apply to URL structure and redirects:

- The root URL is [`https://tai-kun.github.io/surrealdb.js`](https://tai-kun.github.io/surrealdb.js).
- Accessing the root (`/`) will redirect you to the appropriate language page based on your browser's language settings. English is the default. You will also be redirected to the latest version of the documentation.
- Accessing `/<language>` (e.g., `/ja`) will redirect you to the latest version of the documentation written in that language.
- Accessing `/<version>` (e.g., `/v1`) will redirect you to the documentation for the specified version. The language is automatically determined based on your browser settings.
- Accessing `/<language>/<version>` (e.g., `/en/v1`) will display the page for that language and specified version without any redirection.
