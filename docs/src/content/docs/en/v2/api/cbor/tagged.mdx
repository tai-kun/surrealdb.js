`Tagged` is a custom value identified by an integer value from 0 to 2^64-1.

### Import

```ts
import { Tagged } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

```ts
new Tagged<T>(tag: number | BigInt, value: T);
```

#### Arguments

##### `tag`

An integer value from 0 to 2^64-1 to identify the tagged data item. Note that in JavaScript, use BigInt when using numbers exceeding 2^53-1.

##### `value`

The value of the tagged data item.

#### Return Value

When called via `new`, `Tagged` returns its instance.

### `.tag` <Badge variant="instance" /><Badge variant="property" />

`.tag` is a property that holds the integer value from 0 to 2^64-1 to identify the tagged data item.

### `.value` <Badge variant="instance" /><Badge variant="property" />

`.value` is a property that holds the value of the tagged data item.

### `.toCBOR()` <Badge variant="instance" /><Badge variant="method" />

`.toCBOR` is a function that returns information for encoding into CBOR.

```ts
toCBOR(): [tag: number | bigint, value: unknown];
```

#### Arguments

None.

#### Return Value

Returns information for encoding into CBOR.
