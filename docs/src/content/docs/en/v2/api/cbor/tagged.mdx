`Tagged` represents a custom value identified by an integer from 0 to 2<sup>64</sup>-1.

### Import

```ts
import { Tagged } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

```ts
new Tagged<T>(tag: number | BigInt, value: T);
```

#### Arguments

##### `tag`

An integer value from 0 to 2<sup>64</sup>-1 used to identify the tagged data item.  In JavaScript, use `BigInt` for numbers exceeding 2<sup>53</sup>-1.

##### `value`

The value of the tagged data item.

#### Return Value

When called via `new`, `Tagged` returns its instance.

### `.tag` <Badge variant="instance" /><Badge variant="property" />

`.tag` is a property holding an integer value from 0 to 2<sup>64</sup>-1 that identifies the tagged data item.

### `.value` <Badge variant="instance" /><Badge variant="property" />

`.value` is a property storing the value of the tagged data item.

### `.toCBOR()` <Badge variant="instance" /><Badge variant="method" />

`.toCBOR` is a function that returns information for encoding into CBOR.

```ts
toCBOR(): [tag: number | bigint, value: unknown];
```

#### Arguments

None.

#### Return Value

Returns the information for encoding into CBOR.
