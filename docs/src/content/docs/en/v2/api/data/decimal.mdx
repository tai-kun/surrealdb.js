`Decimal` is a class that represents arbitrary precision real numbers. When using the standard preset, this class inherits from [`big.js`](https://github.com/MikeMcl/big.js). This can lead to differences in the results of methods like `.toString()` across presets.

### Import

```ts
import { Decimal } from "@tai-kun/surrealdb/decodeonly-datatypes";
import { Decimal } from "@tai-kun/surrealdb/encodable-datatypes";
import { Decimal } from "@tai-kun/surrealdb/standard-datatypes";
```

### `.constructor()`

Creates a new `Decimal` object.

```ts
new Decimal(source: string); // decode-only, encodable
new Decimal(source: string | number | Decimal); // standard
```

#### Arguments

##### `source`

A real number represented as a string. If the preset is standard, a number can also be provided.

#### Return Value

When called via `new`, `Decimal` returns its instance.

### `.toString()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="decodeonly" />

Gets the real number as a string.

#### Arguments

None.

#### Return Value

Returns a string representation of the real number.

### `.toSurql()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="encodable" />

Converts a `Decimal` object into a string that can be embedded in SurrealQL. Similar to `.toString()` but adds a `dec` suffix to inform the query parser that the string is of type `decimal`.

```ts
.toSurql(): string;
```

#### Arguments

None.

#### Return Value

Returns the real number with a `dec` suffix.

#### Example

```ts
import { Decimal } from "@tai-kun/surrealdb/encodable-datatypes";

const decimal = new Decimal("3.14");
console.log(decimal.toSurql());
//-> 3.14dec
```

### `.clone()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="standard" />

Clones a `Decimal` object.

```ts
.clone(): this;
```

#### Arguments

None.

#### Return Value

Returns a new `Decimal` object. If the `Decimal` class is inherited, an instance of that class is returned.

### Other Methods

If the standard preset is used, the `Decimal` class inherits from the `big.js` `Big` class, so instance methods and properties should refer to the `big.js` documentation:

[`big.js` Documentation](https://mikemcl.github.io/big.js/#prototype-methods)

### Development

#### Generic `Decimal` Object Detection

To determine if a value is a `Decimal` object without relying on the preset, it is recommended to use the `isDecimal` function. This function does not use `instanceof`, making it useful when the preset of the value being validated is unknown.

```ts
import { isDecimal } from "@tai-kun/surrealdb";
```
