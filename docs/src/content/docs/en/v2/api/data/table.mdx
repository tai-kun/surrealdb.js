`Table` is a class that represents a table name with a record ID.

### Import

```ts
import { Table } from "@tai-kun/surrealdb/decodeonly-datatypes";
import { Table } from "@tai-kun/surrealdb/encodable-datatypes";
import { Table } from "@tai-kun/surrealdb/standard-datatypes";
```

### `.constructor()`

Creates a new `Table` object.

```ts
new Table(source: string); // decode-only, encodable
new Table(source: string | { name: string }); // standard
```

#### Arguments

##### `source`

The table name or a `Table` object. If the preset is below standard, only the table name can be passed.

#### Returns

When called through `new`, `Table` returns its instance.

### `.name` <Badge variant="instance" /><Badge variant="property" /><Badge variant="decodeonly" />

The table name. The string is not escaped when retrieving this property.

```ts
.name: string;
```

#### Example

The string is not escaped when retrieving this property.

```ts
import { Table } from "@tai-kun/surrealdb/decodeonly-datatypes";

const table = new Table("foo-bar");
console.log(table.name);
//-> foo-bar
```

### `.toString()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="encodable" />

Converts the `Table` object to a string that can be used in SurrealQL variables.

`.valueOf()` and `.toJSON()` return the same result as this method.

```ts
.toString(): string;
```

#### Arguments

None.

#### Returns

Returns the table name.

#### Example

```ts
import { Table } from "@tai-kun/surrealdb/encodable-datatypes";

const table = new Table("foo-bar");
console.log(table.toString());
//-> foo-bar
```

### `.toSurql()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="encodable" />

Converts the `Table` object to a string that can be embedded in SurrealQL. Similar to `.toString()`, but the table name is escaped as it is in SurrealDB's implementation.

```ts
.toSurql(): string;
```

#### Arguments

None.

#### Returns

Returns the table name as an escaped string.

#### Example

```ts
import { Table } from "@tai-kun/surrealdb/encodable-datatypes";

const table = new Table("foo-bar");
console.log(table.toSurql());
//-> `foo-bar`
```

### `.clone()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="standard" />

Clones the `Table` object.

```ts
.clone(): this;
```

#### Arguments

None.

#### Returns

Returns a new `Table` object. If the `Table` class is inherited, an instance of that class is returned.

### Development

#### Type Guard

You can type-safe the table name of `Table` using a type parameter:

```ts
import { Table } from "@tai-kun/surrealdb/standard-datatypes";

function isUserTable(table: unknown): table is Table<"user"> {
  return table instanceof Table
    && table.name === "user"
}
```

#### Generic Determination of `Table` Object

It is recommended to use the `isTable` function to determine if a value is a `Table` object without relying on the preset. This function does not use `instanceof`, so it is useful when the preset of the value being verified is unknown.

```ts
import { isTable } from "@tai-kun/surrealdb";
```
