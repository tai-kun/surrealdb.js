## quoteStr

The `quoteStr` function quotes a string. For convenience, it quotes empty strings as well.

### Import

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function quoteStr(str: string): string;
```

Quotes a string with double quotes (`"`) or single quotes (`'`).

#### Arguments

##### `str`

The string to be quoted.

#### Returns

Returns the quoted string. The quoting character used depends on whether the given string contains a single quote (`'`) or not:

- If the given string is empty
  - Returns `''`.
- If the given string **does not** contain a single quote
  - Quotes it with single quotes.
- If the given string **does** contain a single quote
  - Quotes it with double quotes.
- If the given string contains both single quotes and double quotes
  - Quotes it with double quotes.
  - Double quotes are escaped with a backslash (`\`).

#### Example

The following example quotes a string **without** a single quote with a single quote:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(``));
console.log(quoteStr(`a`));
console.log(quoteStr(`ğ©¸½`));
console.log(quoteStr(`ğŸ‘ğŸ½`));
console.log(quoteStr(`ãƒã‚š`));
```

Output:

```js
''
'a'
'ğ©¸½'
'ğŸ‘ğŸ½'
'ãƒã‚š'
```

The following example quotes a string **with** a single quote with a double quote:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's`));
console.log(quoteStr(`ğ©¸½'s`));
console.log(quoteStr(`'ğŸ‘ğŸ½`));
console.log(quoteStr(`'ãƒã‚š`));
```

Output:

```js
"cat's"
"ğ©¸½'s"
"'ğŸ‘ğŸ½"
"'ãƒã‚š"
```

The following example quotes a string with both single quotes and double quotes with a double quote and escapes them if necessary:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's "toy"`));
console.log(quoteStr(`'\\"\\`));
console.log(quoteStr(`ğ©¸½'s "feed"`));
console.log(quoteStr(`'ğŸ‘ğŸ½"`));
console.log(quoteStr(`'ãƒã‚š"`));
```

```js

"'\\\"\\"
"ğ©¸½'s \"feed\""
"'ğŸ‘ğŸ½\""
```

## escapeKey

`escapeKey` is similar to [`quoteStr`](/v2/api/utils/quote-str/), but it's used for object keys. For convenience, it quotes empty strings as well.

### Import

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeKey(key: string): string;
```

#### Arguments

##### `key`

The object key.

#### Returns

Returns the escaped string, quoting it with double quotes if necessary:

- If the given string is empty
  - Returns `""`.
- If the given string only contains alphanumeric characters and underscores
  - Returns it as is.
- If the given string contains characters other than alphanumeric characters and underscores
  - Quotes it with double quotes.
- If the given string contains a backslash (`\`)
  - Escapes it.
- If the given string contains a double quote
  - Escapes it.

#### Example

The following example uses a string consisting only of alphanumeric characters and underscores, so you get the same string.

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(`123`));
console.log(escapeKey(`foo_bar`));
```

Output:

```js
123
foo_bar
```

The following example quotes the string with double quotes and escapes them if necessary:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(``));
console.log(escapeKey(`foo-bar`));
console.log(escapeKey(`'\\"\\`));
console.log(escapeKey(`ğ©¸½'s "feed"`));
console.log(escapeKey(`'ğŸ‘ğŸ½"`));
console.log(escapeKey(`'ãƒã‚š"`));
```

Output:

```js
""
"foo-bar"
"'\\"\"
"ğ©¸½'s \"feed\""
"'ğŸ‘ğŸ½\""
"'ãƒã‚š\""
```
