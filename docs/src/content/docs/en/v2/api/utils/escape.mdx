## quoteStr

`quoteStr` is a function that encloses a string with quotation marks. For convenience, even an empty string is enclosed in quotation marks.

### Import

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function quoteStr(str: string): string;
```

Encloses a string with quotation marks (`"` or `'`).

#### Arguments

##### `str`

The string to be enclosed in quotation marks.

#### Return Value

Returns the string enclosed in quotation marks. The quotation marks used for enclosing depend on whether the given string contains a single quotation mark (`'`) or not:

- If the given string is empty
  - Returns `''`.
- If the given string does **not** contain a single quotation mark
  - Encloses it with single quotation marks.
- If the given string does **contain** a single quotation mark
  - Encloses it with double quotation marks.
- If the given string contains both single and double quotation marks
  - Encloses it with double quotation marks.
  - Escapes double quotation marks with a backslash (`\`)

#### Examples

The following examples enclose strings that **do not** contain single quotation marks with single quotation marks:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(``));
console.log(quoteStr(`a`));
console.log(quoteStr(`ğ©¸½`));
console.log(quoteStr(`ğŸ‘ğŸ½`));
console.log(quoteStr(`ãƒã‚š`));
```

Output:

```js
''
'a'
'ğ©¸½'
'ğŸ‘ğŸ½'
'ãƒã‚š'
```

The following examples enclose strings that **do** contain single quotation marks with double quotation marks:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's`));
console.log(quoteStr(`ğ©¸½'s`));
console.log(quoteStr(`'ğŸ‘ğŸ½`));
console.log(quoteStr(`'ãƒã‚š`));
```

Output:

```js
"cat's"
"ğ©¸½'s"
"'ğŸ‘ğŸ½"
"'ãƒã‚š"
```

The following examples enclose strings that contain both single and double quotation marks with double quotation marks and escape them if necessary:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's "toy"`));
console.log(quoteStr(`'\\"\\`));
console.log(quoteStr(`ğ©¸½'s "feed"`));
console.log(quoteStr(`'ğŸ‘ğŸ½"`));
console.log(quoteStr(`'ãƒã‚š"`));
```

## escapeKey

`escapeKey` is similar to [`quoteStr`](/v2/api/utils/quote-str/) but is a function used for object keys. For convenience, even an empty string is enclosed in quotation marks.

### Import

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeKey(key: string): string;
```

#### Arguments

##### `key`

The object key.

#### Return Value

Returns the escaped string. Encloses it in double quotation marks if necessary:

- If the given string is empty
  - Returns `""`.
- If the given string does not start with a number and consists only of alphanumeric characters and underscores
  - Returns it as is.
- If the given string contains a double quotation mark
  - Escapes it.
- If the given string contains a backslash (`\`)
  - Escapes it.
- If the given string starts with a number or contains characters other than alphanumeric characters and underscores
  - Encloses it in double quotation marks.

#### Examples

The following example shows how to obtain the string as it is because the string consists only of alphanumeric characters and underscores:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(`foo_bar`));
```

Output:

```js
foo_bar
```

The following examples enclose the string in double quotation marks and escape them if necessary:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(``));
console.log(escapeKey(`123`));
console.log(escapeKey(`foo-bar`));
console.log(escapeKey(`'\\"\\`));
console.log(escapeKey(`ğ©¸½'s "feed"`));
console.log(escapeKey(`'ğŸ‘ğŸ½"`));
console.log(escapeKey(`'ãƒã‚š"`));
```

Output:

```js
""
"123"
"foo-bar"
"'\\\"\\"
"ğ©¸½'s \"feed\""
"'ğŸ‘ğŸ½\""
"'ãƒã‚š\""
```
