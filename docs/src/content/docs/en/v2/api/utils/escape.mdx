## quoteStr

`quoteStr` is a function that encloses a string with quotation marks. For convenience, even an empty string is enclosed in quotation marks.

### Import

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function quoteStr(str: string): string;
```

Encloses a string with quotation marks (`"` or `'`).

#### Arguments

##### `str`

The string to be enclosed in quotation marks.

#### Return Value

Returns the string enclosed in quotation marks. The quotation marks used for enclosing depend on whether the given string contains a single quotation mark (`'`) or not:

- If the given string is empty
  - Returns `''`.
- If the given string does **not** contain a single quotation mark
  - Encloses it with single quotation marks.
- If the given string does **contain** a single quotation mark
  - Encloses it with double quotation marks.
- If the given string contains both single and double quotation marks
  - Encloses it with double quotation marks.
  - Escapes double quotation marks with a backslash (`\`)

#### Examples

The following examples enclose strings that **do not** contain single quotation marks with single quotation marks:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(``));
console.log(quoteStr(`a`));
console.log(quoteStr(`𩸽`));
console.log(quoteStr(`👍🏽`));
console.log(quoteStr(`パ`));
```

Output:

```js
''
'a'
'𩸽'
'👍🏽'
'パ'
```

The following examples enclose strings that **do** contain single quotation marks with double quotation marks:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's`));
console.log(quoteStr(`𩸽's`));
console.log(quoteStr(`'👍🏽`));
console.log(quoteStr(`'パ`));
```

Output:

```js
"cat's"
"𩸽's"
"'👍🏽"
"'パ"
```

The following examples enclose strings that contain both single and double quotation marks with double quotation marks and escape them if necessary:

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's "toy"`));
console.log(quoteStr(`'\\"\\`));
console.log(quoteStr(`𩸽's "feed"`));
console.log(quoteStr(`'👍🏽"`));
console.log(quoteStr(`'パ"`));
```

Output:

```js
"cat's \"toy\""
"'\\\"\\"
"𩸽's \"feed\""
"'👍🏽\""
"'パ\""
```

## escapeKey

`escapeKey` is similar to [`quoteStr`](/v2/api/utils/quote-str/) but is a function used for object keys. For convenience, even an empty string is enclosed in quotation marks.

### Import

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeKey(key: string): string;
```

#### Arguments

##### `key`

The object key.

#### Return Value

Returns the escaped string. Encloses it in double quotation marks if necessary:

- If the given string is empty
  - Returns `""`.
- If the given string does not start with a number and consists only of alphanumeric characters and underscores
  - Returns it as is.
- If the given string contains a double quotation mark
  - Escapes it.
- If the given string contains a backslash (`\`)
  - Escapes it.
- If the given string starts with a number or contains characters other than alphanumeric characters and underscores
  - Encloses it in double quotation marks.

#### Examples

The following example shows how to obtain the string as it is because the string consists only of alphanumeric characters and underscores:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(`foo_bar`));
```

Output:

```js
foo_bar
```

The following examples enclose the string in double quotation marks and escape them if necessary:

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(``));
console.log(escapeKey(`123`));
console.log(escapeKey(`foo-bar`));
console.log(escapeKey(`'\\"\\`));
console.log(escapeKey(`𩸽's "feed"`));
console.log(escapeKey(`'👍🏽"`));
console.log(escapeKey(`'パ"`));
```

Output:

```js
""
"123"
"foo-bar"
"'\\\"\\"
"𩸽's \"feed\""
"'👍🏽\""
"'パ\""
```

## escapeRid

`escapeRid` is similar to [`quoteStr`](/v2/api/utils/quote-str/) but is a function used for record IDs. For convenience, even an empty string is enclosed in quotation marks.

### Import

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeRid(rid: string): string;
```

#### Arguments

##### `rid`

The ID part of a record ID.

#### Return Value

Returns the escaped string. Encloses it in brackets (`⟨⟩`) if necessary:

- If the given string is empty
  - Returns `⟨⟩`.
- If the given string does not start with a number and consists only of alphanumeric characters and underscores
  - Returns it as is.
- If the given string contains a bracket (`⟩`)
  - Escapes it.
- If the given string consists entirely of numbers or contains characters other than alphanumeric characters and underscores
  - Encloses it in brackets (`⟨⟩`).

#### Examples

The following example shows how to obtain the string as it is because the string does not start with a number and consists only of alphanumeric characters and underscores

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";

console.log(escapeRid(`foo_123`));
```

Output:

```js
foo_123
```

The following examples enclose the string in brackets (`⟨⟩`) and escape them if necessary:

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";

console.log(escapeRid(``));
console.log(escapeRid(`123`));
console.log(escapeRid(`foo-bar`));
console.log(escapeRid(`𩸽's ⟨feed⟩`));
console.log(escapeRid(`'👍🏽⟩`));
console.log(escapeRid(`'パ⟩`));
```

Output:

```js
⟨⟩
⟨123⟩
⟨foo-bar⟩
⟨𩸽's ⟨feed\⟩⟩
⟨'👍🏽\⟩⟩
⟨'パ\⟩⟩
```

## escapeIdent

`escapeIdent` is similar to [`quoteStr`](/v2/api/utils/quote-str/) but is a function used for namespaces, databases, tables, and field names. For convenience, even an empty string is enclosed in quotation marks.

### Import

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function escapeIdent(ident: string): string;
```

#### Arguments

##### `ident`

The identifier.

#### Return Value

Returns the escaped string. Encloses it in backticks (<code>`</code>) if necessary:

- If the given string is empty
  - Returns <code>``</code>. {/* TODO(tai-kun): " になる問題解決する */}
- If the given string does not start with a number and consists only of alphanumeric characters and underscores
  - Returns it as is.
- If the given string contains a backtick (<code>`</code>)
  - Escapes it.
- If the given string starts with a number or contains characters other than alphanumeric characters and underscores
  - Encloses it in backticks (<code>`</code>).

#### Examples

The following example shows how to obtain the string as it is because the string does not start with a number and consists only of alphanumeric characters and underscores

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";

console.log(escapeIdent(`foo_123`));
```

Output:

```js
foo_123
```

The following examples enclose the string in backticks (<code>`</code>) and escape them if necessary:

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";

console.log(escapeIdent(""));
console.log(escapeIdent("123"));
console.log(escapeIdent("foo-bar"));
console.log(escapeIdent("𩸽's `feed`"));
console.log(escapeIdent("'👍🏽`"));
console.log(escapeIdent("'パ`"));
```

Output:

```js
``
`123`
`foo-bar`
`𩸽's \`feed\``
`'👍🏽\``
`'パ\``
```

### String Length

The strings used in the previous examples contain characters whose length as seen by JavaScript differs from their length as seen by humans:

| Type | Display | `.length` | `Uint8Array` |
| --- | --- | --- | --- |
| ASCII | a | 1 | `[ 0x61 ]` |
| Surrogate Pair | 𩸽 | 2 | `[ 0xF0, 0xA9, 0xB8, 0xBD ]` |
| Variation Selector | 👍🏽 | 4 | `[ 0xF0, 0x9F, 0x91, 0x8D, 0xF0, 0x9F, 0x8F, 0xBD ]` |
| Combining Character | パ | 2 | `[ 0xE3, 0x83, 0x8F, 0xE3, 0x82, 0x9A ]` |

If the value of `.length` is counterintuitive, it is necessary to carefully consider operations on strings.
