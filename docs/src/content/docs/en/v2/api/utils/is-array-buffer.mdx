The `isArrayBuffer` function determines whether the provided value is an `ArrayBuffer` object.

### Import

```ts
import { isArrayBuffer } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function isArrayBuffer(value: unknown): value is ArrayBuffer;
```

#### Arguments

##### `value`

The value expected to be an `ArrayBuffer` object.

#### Return Value

Returns `true` if `value` is an `ArrayBuffer` object, otherwise `false`.

### Explanation

In some versions of Firefox, `fetch` responses with an `ArrayBuffer` object might have methods but fail `instanceof` checks. Therefore, the function utilizes tag name verification for determination. This approach is similar to the behavior of lodash.
