`isSafeNumber` verifies if a given number is safe for arithmetic operations.

### Import

```ts
import { isSafeNumber } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function isSafeNumber(value: number): boolean;
```

#### Argument

##### `value`

The number to be verified.
