The `makeAbortApi` function creates an abort signal and an associated abort function.

### Import

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function makeAbortApi(signal?: AbortSignal): [
  signal: AbortSignal,
  abort: (reason?: unknown) => void,
];
```

#### Arguments

##### `signal`

The abort signal to associate with the abort API.

#### Return Value

The function returns an array with the following elements in order:

##### `signal`

The abort signal. The signal is not initially aborted.

##### `abort`

Dispatches an `"abort"` event to the abort signal. Optionally accepts a single argument, `reason`, which is sent as the reason for the abort via the `"abort"` event.

#### Examples

The following example shows how to create an Abort API without arguments:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const [signal, abort] = makeAbortApi();
abort();
```

The above example is functionally equivalent to `new AbortController()`, except that `makeAbortApi` returns an array instead of an object.

The following example shows how to create an Abort API with an associated abort signal:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

controller.abort("test");
```

Output:

```js
controller.signal test
signal test
```

Abort events propagate from the Abort API to the associated abort signal:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal, abort] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

abort("test");
```

Output:

```js
controller.signal test
signal test
```
