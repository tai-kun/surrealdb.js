`throwIfAborted` is a function that throws an error if the provided abort signal is aborted, including the reason for the abortion. This function is provided for environments that do not have `new AbortSignal().throwIfAborted()` implemented.

## Importing

```ts
import { throwIfAborted } from "@tai-kun/surrealdb/utils";
```

## Usage

```ts
function throwIfAborted(signal: AbortSignal | null | undefined): void;
```

#### Arguments

`signal`
:   The abort signal.

#### Return value

None.
