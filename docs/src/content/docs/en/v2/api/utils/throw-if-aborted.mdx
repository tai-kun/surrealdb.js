The `throwIfAborted` function checks if the provided abort signal has been aborted. If it has, it throws an error with the reason for the abort. This function is provided for environments that do not natively support `new AbortSignal().throwIfAborted()`.

## Import

```ts
import { throwIfAborted } from "@tai-kun/surrealdb/utils";
```

## Usage

```ts
function throwIfAborted(signal: AbortSignal | null | undefined): void;
```

#### Arguments

`signal`
: The abort signal to check.

#### Return Value

None.
