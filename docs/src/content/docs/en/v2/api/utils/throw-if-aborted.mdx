`throwIfAborted` is a function that throws an error if the provided AbortSignal has been aborted. It's provided for environments that don't natively implement `new AbortSignal().throwIfAborted()`.

## Importing

```ts
import { throwIfAborted } from "@tai-kun/surrealdb/utils";
```

## Usage

```ts
function throwIfAborted(signal: AbortSignal | null | undefined): void;
```

#### Arguments

`signal`
:   The AbortSignal.


#### Return Value

None.
