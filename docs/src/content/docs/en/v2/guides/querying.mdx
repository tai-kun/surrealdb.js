### `.query()` <Badge variant="instance" /><Badge variant="method" />

Executes custom SurrealQL queries.

```ts
query(
  surql: string | object,
  vars?: { [p: string]: unknown },
  options?: ClientRpcOptions,
): Promise<unknown[]>;
```

#### Parameters

##### `surql`

A string containing the SurrealQL query or a pre-prepared query object.

##### `vars`

Variables to pass to the SurrealQL query at execution time. Note that variable names should not start with `$`.

##### `options`

Options for the PRC request.

#### Return Value

Returns a `Promise` object that resolves with an array of results from the SurrealQL query execution. If one or more statements fail, it rejects with a `QueryFailedError` containing the error messages.

Note that there is no way to retrieve the execution results if this method rejects. If you need to handle errors manually, you can use [`.queryRaw()`](#queryraw).

#### Examples

The following example shows a successful query execution:

```ts
import { Surreal } from "@tai-kun/surrealdb";

const db = new Surreal();
await db.connect("ws://localhost:8000");

const results = await db.query(`
  CREATE ONLY user:foo CONTENT { age: 42 };
`);
console.log(results);
// [{
//   id: Thing { tb: 'user', id: 'foo' },
//   age: 42
// }]

await db.close();
```

The following example throws a `QueryFailedError` because one of the statements fails:

```ts
import { Surreal } from "@tai-kun/surrealdb";

const db = new Surreal();
await db.connect("ws://localhost:8000");

const results = await db.query(`
  CREATE ONLY user:foo CONTENT { age: 42 };
  OUTPUT 42;
`); // throws QueryFailedError: Query failed with 1 error(s)
```

The following example executes a pre-prepared query:

```ts
import { surql, Surreal, Thing } from "@tai-kun/surrealdb";

const CreateUserQuery = surql`
  CREATE ONLY user:foo CONTENT { age: 42 };
`.returns<[{ id: Thing<"user">, age: number }]>()

const db = new Surreal();
await db.connect("ws://localhost:8000");

const results = await db.query(CreateUserQuery);
   // ^? const results: [{ id: Thing<"user">, age: number }]

await db.close();
```

### `.queryRaw()` <Badge variant="instance" /><Badge variant="method" />

Executes custom SurrealQL queries.

```ts
queryRaw(
  surql: string | object,
  vars?: { [p: string]: unknown },
  options?: ClientRpcOptions,
): Promise<QueryResult[]>;
```

#### Parameters

##### `surql`

A string containing the SurrealQL query or a pre-prepared query object.

##### `vars`

Variables to pass to the SurrealQL query at execution time. Note that variable names should not start with `$`.

##### `options`

Options for the PRC request.

#### Return Value

Returns a `Promise` object that resolves with an array of results from the SurrealQL query execution.

```ts
type QueryResult = {
  status: "OK";
  time: string;
  result: unknown;
} | {
  status: "ERR";
  time: string;
  result: string;
};
```
