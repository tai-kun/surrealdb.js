---
title: makeAbortApi
---

`makeAbortApi` is a function that creates an abort signal and an associated abort function.

### Import

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function makeAbortApi(signal?: AbortSignal): [
  signal: AbortSignal,
  abort: (reason?: unknown) => void,
];
```

#### Arguments

##### `signal`

The abort signal to associate with.

#### Return Value

Returns an array with the following elements in order:

##### `signal`

The abort signal. It is not initially aborted.

##### `abort`

Dispatches an `"abort"` event to the abort signal. It optionally accepts a single argument `reason`, which can be used to communicate the reason for the abort through the `"abort"` event.

#### Examples

The following example creates an Abort API without any arguments:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const [signal, abort] = makeAbortApi();
abort();
```

The above example is almost equivalent to `new AbortController()`. The difference is that `makeAbortApi` returns an array instead of an object.

The following example creates an Abort API with an associated abort signal:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

controller.abort("test");
```

Output:

```js
controller.signal test
signal test
```

The abort event propagates from the Abort API to the associated abort signal even if it is aborted by the Abort API:

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal, abort] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

abort("test");
```

Output:

```js
controller.signal test
signal test
```
