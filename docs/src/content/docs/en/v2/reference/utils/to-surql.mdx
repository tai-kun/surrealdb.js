---
title: toSurql
---

The `toSurql` function converts JavaScript values into SurrealQL strings.

### Import

```ts
import { toSurql } from "@tai-kun/surrealdb/utils";
```

### Usage

```ts
function toSurql(value: unknown): string;
```

#### Arguments

##### `value`

This argument represents a JavaScript value. It accepts any of the following types:

- `string`
- `number`
- `bigint`
- `boolean`
- `undefined`
- `null`
- `{ toSurql(): string }`
- `Date`
- `{ toJSON(): string }`
- `unknown[]`
- `{ [key: string]: unknown }`
- `Set<unknown>`
- `Map<unknown, unknown>`

`unknown` encompasses values of the above types.

The conversion prioritizes these types in the order listed. This means that if a plain object has both `.toJSON` and `.toSurql` functions, the latter takes precedence.

For plain object identification, the [`is-plain-obj`](https://github.com/sindresorhus/is-plain-obj) library is utilized. Consequently, if a plain object lacks both `.toSurql` and `.toJSON` functions while having `Symbol.toStringTag` or `Symbol.iterator`, it will not be classified as an object and an error will be thrown.

#### Return Value

The function returns a SurrealQL string.

#### Example

```ts
import { toSurql } from "@tai-kun/surrealdb/utils";

const surql = toSurql({
  bigint: 9007199254740992n, // Number.MAX_SAFE_INTEGER + 1
  boolean: [true, false],
  date: new Date("2024-06-01T12:34:56.789Z"),
  map: new Map([
    [0, {
      toJSON: () => [0, 1],
    }],
  ]),
  null: null,
  number: 123,
  set: new Set([{
    toSurql: () => `<bytes>"hello"`,
  }]),
  string: "文字列",
  undefined: undefined,
});

console.log(surql);
```

Output:

```text
{bigint:9007199254740992,boolean:[true,false],date:d'2024-06-01T12:34:56.789Z',map:{0:[0,1]},null:NULL,number:123,set:[<bytes>"hello"],string:'文字列',undefined:NONE}
```
