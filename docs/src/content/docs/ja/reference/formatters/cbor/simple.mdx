`Simple` は 0 から 255 までの整数値で識別される単純な値です。

### インポート

```ts
import { Simple } from "@tai-kun/surreal/formatters/cbor";
```

### constructor

新しい `Simple` オブジェクトを作成します。

```ts
new Simple(value: number);
```

#### 引数

##### `value`

0 から 255 までの整数値です。

#### 返値

`new` を通じて呼び出された場合、`Simple` はそのインスタンスを返します。

### .value <Badge variant="instance" /><Badge variant="property" />

`Simple` オブジェクトを識別する 0 から 255 までの整数値です。
