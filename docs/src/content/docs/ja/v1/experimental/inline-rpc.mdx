インライン RPC は `Surreal` クラスのインスタンスを作成しなくとも、その場で 1 回限りの RPC を実行する関数です。使い方は `.rpc()` メソッドとほとんど同じです。ただしデフォルトでは `JsonFormatter` が使用されていることに注意してください。

### インポート

```ts
import { rpc } from "@tai-kun/surrealdb";
```

### 例

```ts
import { rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});
```
