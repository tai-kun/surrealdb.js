## バージョン

このドキュメントでは [`@tai-kun/surrealdb`](https://www.npmjs.com/package/@tai-kun/surrealdb) の使用方法をメジャーバージョンごとに確認できます。

- v2: [`/v2`](https://tai-kun.github.io/surrealdb.js/v2) (最新バージョン)
- v1: [`/v1`](https://tai-kun.github.io/surrealdb.js/v1)

## 国際化

このドキュメントは以下の言語に対応しています。

- 日本語: [`/ja`](https://tai-kun.github.io/surrealdb.js/ja) (オリジナル)
- 中国語: [`/zh-CN`](https://tai-kun.github.io/surrealdb.js/zh-CN)
- 韓国語: [`/ko`](https://tai-kun.github.io/surrealdb.js/ko)
- 英語: [`/en`](https://tai-kun.github.io/surrealdb.js/en)

日本語以外の言語は、機械翻訳によって生成されています。翻訳内容には十分注意していますが、誤訳や不適切な表現が含まれている場合は、意図的なものではありませんのでご了承ください。

## リダイレクト

URL 構成およびリダイレクトのルールは以下の通りです。

- ルート URL は [`https://tai-kun.github.io/surrealdb.js`](https://tai-kun.github.io/surrealdb.js) です。
- ルート (`/`) にアクセスすると、ブラウザの言語設定に基づき適切な言語ページにリダイレクトされます。デフォルトは英語です。また、最新バージョンのドキュメントへリダイレクトされます。
- `/<language>` (例: `/ja`) にアクセスすると、その言語で書かれた最新バージョンのドキュメントにリダイレクトされます。
- `/<version>` (例: `/v1`) にアクセスすると、指定したバージョンのドキュメントにリダイレクトされます。言語はブラウザの設定に基づいて自動判定されます。
- `/<language>/<version>` (例: `/ja/v1`) にアクセスすると、その言語かつ指定バージョンのページが表示され、リダイレクトは行われません。
