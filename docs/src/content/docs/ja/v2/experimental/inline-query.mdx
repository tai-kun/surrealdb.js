インラインクエリーは `Surreal` クラスのインスタンスを作成しなくとも、その場で 1 回限りのカスタムクエリーを実行する関数です。使い方は [`.query()`](/v2/guides/querying#query) メソッドとほとんど同じです。ただしデフォルトでは `JsonFormatter` が使用されていることに注意してください。

### インポート

```ts
import { query } from "@tai-kun/surrealdb";
```

### 例

```ts
import { query, rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});

const results = await query("http://127.0.0.1:8000", "RETURN 42", {}, {
  token,
  namespace: "test",
  database: "test",
});
```
