---
title: isSafeNumber
---

`isSafeNumber` は引数に渡された数値が、演算上安全な数値であるかを検証します。

### インポート

```ts
import { isSafeNumber } from "@tai-kun/surrealdb/utils";
```

### 使い方

```ts
function isSafeNumber(value: number): boolean;
```

#### 引数

##### `value`

検証する数値です。

#### 返値

`value` が次の場合は `false` を返します:

- `NaN`
- 無限
- ±2^53-1 の範囲を超える
