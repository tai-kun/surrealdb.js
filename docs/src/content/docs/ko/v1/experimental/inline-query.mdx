인라인 쿼리는 `Surreal` 클래스 인스턴스를 생성하지 않고도, 필요한 시점에 한 번만 사용하는 사용자 정의 쿼리를 실행하는 함수입니다.  사용법은 [`.query()`](/ko/guides/querying#query) 메서드와 거의 동일합니다. 단, 기본적으로 `JsonFormatter`가 사용된다는 점에 유의하십시오.

### import

```ts
import { query } from "@tai-kun/surrealdb";
```

### 예시

```ts
import { query, rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});

const results = await query("http://127.0.0.1:8000", "RETURN 42", {}, {
  token,
  namespace: "test",
  database: "test",
});
```
