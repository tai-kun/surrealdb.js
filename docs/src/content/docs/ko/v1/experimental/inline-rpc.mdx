인라인 RPC는 `Surreal` 클래스의 인스턴스를 만들지 않고도 즉석에서 일회성 RPC를 실행하는 함수입니다. 사용법은 `.rpc()` 메서드와 거의 동일합니다. 그러나 기본적으로 `JsonFormatter`가 사용된다는 점에 유의하십시오.

### 임포트

```ts
import { rpc } from "@tai-kun/surrealdb";
```

### 예시

```ts
import { rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});
```
