`Tagged`는 0부터 2^64-1까지의 정수 값으로 식별되는 사용자 정의 값입니다.

### import

```ts
import { Tagged } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

```ts
new Tagged<T>(tag: number | BigInt, value: T);
```

#### 매개변수

##### `tag`

태그가 지정된 데이터 항목을 식별하기 위한 0부터 2^64-1까지의 정수 값입니다. JavaScript에서는 2^53-1을 초과하는 숫자를 사용하는 경우 BigInt를 사용하십시오.

##### `value`

태그가 지정된 데이터 항목의 값입니다.

#### 반환값

`new`를 통해 호출된 경우 `Tagged`는 해당 인스턴스를 반환합니다.

### `.tag` <Badge variant="instance" /><Badge variant="property" />

`.tag`는 태그가 지정된 데이터 항목을 식별하기 위한 0부터 2^64-1까지의 정수 값을 갖는 속성입니다.

### `.value` <Badge variant="instance" /><Badge variant="property" />

`.value`는 태그가 지정된 데이터 항목의 값을 저장하는 속성입니다.

### `.toCBOR()` <Badge variant="instance" /><Badge variant="method" />

`.toCBOR`은 CBOR로 인코딩하기 위한 정보를 반환하는 함수입니다.

```ts
toCBOR(): [tag: number | bigint, value: unknown];
```

#### 매개변수

없음.

#### 반환값

CBOR로 인코딩하기 위한 정보를 반환합니다.
