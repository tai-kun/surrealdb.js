`makeAbortApi`는 취소 신호와 관련된 취소 함수를 만드는 함수입니다.

### 가져오기

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";
```

### 사용법

```ts
function makeAbortApi(signal?: AbortSignal): [
  signal: AbortSignal,
  abort: (reason?: unknown) => void,
];
```

#### 인수

##### `signal`

연결할 취소 신호입니다.

#### 반환 값

다음 순서로 정렬된 배열을 반환합니다.

##### `signal`

취소 신호입니다. 초기 상태에서는 취소되지 않았습니다.

##### `abort`

취소 신호에 `"abort"` 이벤트를 전송합니다. 옵션으로 `reason` 인수를 하나 받을 수 있으며, `"abort"` 이벤트를 통해 취소 사유를 전달할 수 있습니다.

#### 예시

다음 예시는 인수 없이 Abort API를 만듭니다.

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const [signal, abort] = makeAbortApi();
abort();
```

위 예시는 `new AbortController()`와 거의 동일한 의미를 가집니다. `AbortController`와 다른 점은 `makeAbortApi`가 객체가 아니라 배열을 반환한다는 것입니다.

다음 예시는 연결할 취소 신호를 지정하여 Abort API를 만듭니다.

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

controller.abort("test");
```

출력:

```js
controller.signal test
signal test
```

취소 이벤트는 Abort API에서 취소해도 연결된 취소 신호로 전파됩니다.

```ts
import { makeAbortApi } from "@tai-kun/surrealdb/utils";

const controller = new AbortController();
controller.signal.addEventListener("abort", function() {
  console.log("controller.signal", this.reason);
});

const [signal, abort] = makeAbortApi(controller.signal);
signal.addEventListener("abort", function() {
  console.log("signal", this.reason);
});

abort("test");
```

출력:

```js
controller.signal test
signal test
```
