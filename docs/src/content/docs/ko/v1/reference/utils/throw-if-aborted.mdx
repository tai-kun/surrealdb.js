`throwIfAborted` 함수는 전달된 중단 신호가 중단된 경우 그 이유를 throw하는 함수입니다. `new AbortSignal().throwIfAborted()`가 구현되지 않은 환경을 위해 제공됩니다.

## import

```typescript
import { throwIfAborted } from "@tai-kun/surrealdb/utils";
```

## 사용법

```typescript
function throwIfAborted(signal: AbortSignal | null | undefined): void;
```

#### 매개변수

`signal`
: 중단 신호입니다.

#### 반환값

없음.
