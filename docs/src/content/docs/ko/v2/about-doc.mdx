## 버전

이 문서에서는 [`@tai-kun/surrealdb`](https://www.npmjs.com/package/@tai-kun/surrealdb) 사용 방법을 메이저 버전별로 확인할 수 있습니다.

- v2: [`/v2`](https://tai-kun.github.io/surrealdb.js/v2) (최신 버전)
- v1: [`/v1`](https://tai-kun.github.io/surrealdb.js/v1)

## 다국어 지원

이 문서는 다음 언어를 지원합니다.

- 한국어: [`/ko`](https://tai-kun.github.io/surrealdb.js/ko)
- 일본어: [`/ja`](https://tai-kun.github.io/surrealdb.js/ja) (원본)
- 중국어: [`/zh-CN`](https://tai-kun.github.io/surrealdb.js/zh-CN)
- 영어: [`/en`](https://tai-kun.github.io/surrealdb.js/en)

한국어를 제외한 언어는 기계 번역으로 생성되었습니다. 번역 내용에는 최대한 주의를 기울였지만, 오역이나 부적절한 표현이 포함되어 있는 경우 의도적인 것이 아니므로 양해 부탁드립니다.


## 리다이렉트

URL 구성 및 리다이렉트 규칙은 다음과 같습니다.

- 루트 URL은 [`https://tai-kun.github.io/surrealdb.js`](https://tai-kun.github.io/surrealdb.js) 입니다.
- 루트 (`/`)에 접근하면 브라우저의 언어 설정에 따라 적절한 언어 페이지로 리다이렉트됩니다. 기본값은 영어이며, 최신 버전의 문서로 리다이렉트됩니다.
- `/<language>` (예: `/ja`)에 접근하면 해당 언어로 작성된 최신 버전의 문서로 리다이렉트됩니다.
- `/<version>` (예: `/v1`)에 접근하면 지정된 버전의 문서로 리다이렉트됩니다. 언어는 브라우저 설정에 따라 자동으로 판별됩니다.
- `/<language>/<version>` (예: `/ko/v1`)에 접근하면 해당 언어 및 지정된 버전의 페이지가 표시되며 리다이렉트는 수행되지 않습니다.
