`isArrayBuffer` 함수는 입력으로 주어진 값이 `ArrayBuffer` 객체인지 확인합니다.

### 가져오기

```ts
import { isArrayBuffer } from "@tai-kun/surrealdb/utils";
```

### 사용법

```ts
function isArrayBuffer(value: unknown): value is ArrayBuffer;
```

#### 인수

##### `value`

`ArrayBuffer` 객체를 기대하는 값입니다.

#### 반환값

`value`가 `ArrayBuffer` 객체이면 `true`를 반환하고, 그렇지 않으면 `false`를 반환합니다.

### 설명

Firefox의 버전에 따라 `fetch`의 응답 본문을 `ArrayBuffer` 객체로 처리할 경우, `instanceof`로 판별이 불가능한 경우가 발생할 수 있습니다. 따라서 태그 이름으로 판별합니다. 이는 lodash의 동작과 유사합니다.
