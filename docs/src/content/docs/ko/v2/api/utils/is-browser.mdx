`isBrowser`는 현재 실행되고 있는 JavaScript 런타임이 브라우저 환경인지 판별하는 함수입니다.

### 가져오기

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";
```

### 사용 방법

```ts
function isBrowser(): boolean;
```

#### 인수

없음.

#### 반환 값

브라우저 환경이면 `true`를 반환하고, 그렇지 않으면 `false`를 반환합니다.

#### 예

다음 예에서는 브라우저 환경인지 판별하는 방법을 보여줍니다:

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";

if (isBrowser()) {
  document;
}
```

### 설명

전역 변수 `document`가 `undefined`가 아니면 브라우저로 판단합니다. 예를 들어 Deno에서는 `window` 객체를 전역으로 추가하기 때문에, `window` 객체의 유무로 판별하면 서버 런타임에서도 브라우저로 잘못 판단할 수 있습니다. 웹 프레임워크 Remix는 이를 [Document Guard](https://remix.run/docs/en/main/guides/constraints#document-guard)라고 부릅니다.
