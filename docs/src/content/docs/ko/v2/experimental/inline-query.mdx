인라인 쿼리는 `Surreal` 클래스의 인스턴스를 만들지 않고도 그 자리에서 한 번만 사용할 수 있는 사용자 지정 쿼리를 실행하는 함수입니다. 사용 방법은 [`.query()`](/v2/guides/querying#query) 메서드와 거의 동일합니다. 단, 기본적으로 `JsonFormatter`가 사용된다는 점에 유의하세요.

### 임포트

```ts
import { query } from "@tai-kun/surrealdb";
```

### 예시

```ts
import { query, rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});

const results = await query("http://127.0.0.1:8000", "RETURN 42", {}, {
  token,
  namespace: "test",
  database: "test",
});
```
