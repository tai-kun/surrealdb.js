인라인 RPC는 `Surreal` 클래스의 인스턴스를 생성하지 않고도 바로 한 번의 RPC를 실행할 수 있는 함수입니다. 사용법은 `.rpc()` 메서드와 거의 동일하지만, 기본적으로 `JsonFormatter`가 사용된다는 점에 유의하세요.

### 가져오기

```ts
import { rpc } from "@tai-kun/surrealdb";
```

### 예시

```ts
import { rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});
```
