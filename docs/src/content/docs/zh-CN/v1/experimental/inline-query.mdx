内联查询是一个函数，无需创建 `Surreal` 类实例，即可在原地执行一次性的自定义查询。使用方法与 [`.query()`](/zh-CN/guides/querying#query) 方法几乎相同，但请注意，默认情况下会使用 `JsonFormatter`。

### 导入

```ts
import { query } from "@tai-kun/surrealdb";
```

### 例子

```ts
import { query, rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});

const results = await query("http://127.0.0.1:8000", "RETURN 42", {}, {
  token,
  namespace: "test",
  database: "test",
});
```
