内联 RPC 是一个函数，它允许您在不创建 `Surreal` 类实例的情况下立即执行一次性 RPC。它的使用方法与 `.rpc()` 方法几乎相同。但请注意，默认情况下它使用的是 `JsonFormatter`。

### 导入

```ts
import { rpc } from "@tai-kun/surrealdb";
```

### 示例

```ts
import { rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});
```
