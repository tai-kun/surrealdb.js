### `.query()` <Badge variant="instance" /><Badge variant="method" />

执行自定义 SurrealQL。

```ts
query(
  surql: string | object,
  vars?: { [p: string]: unknown },
  options?: ClientRpcOptions,
): Promise<unknown[]>;
```

#### 参数

##### `surql`

SurrealQL 字符串或预先准备好的查询。

##### `vars`

在执行 SurrealQL 时传递的变量。请注意，变量名不要以美元符号开头。

##### `options`

PRC 请求的选项。

#### 返回值

返回一个以 SurrealQL 执行结果的数组为解析结果的 `Promise` 对象。如果语句出现一个或多个失败，则会以包含这些错误消息的 `QueryFailedError` 拒绝。

如果此方法被拒绝，则无法获得执行结果。如果需要手动进行错误处理，可以使用 [`.queryRaw()`](#queryraw)。

#### 例子

以下示例成功执行了查询：

```ts
import { Surreal } from "@tai-kun/surrealdb";

const db = new Surreal();
await db.connect("ws://localhost:8000");

const results = await db.query(`
  CREATE ONLY user:foo CONTENT { age: 42 };
`);
console.log(results);
// [{
//   id: Thing { tb: 'user', id: 'foo' },
//   age: 42
// }]

await db.disconnect();
```

以下示例中，其中一个语句失败，因此查询将以 `QueryFailedError` 拒绝：

```ts
import { Surreal } from "@tai-kun/surrealdb";

const db = new Surreal();
await db.connect("ws://localhost:8000");

const results = await db.query(`
  CREATE ONLY user:foo CONTENT { age: 42 };
  OUTPUT 42;
`); // throws QueryFailedError: Query failed with 1 error(s)
```

以下示例执行预先准备好的查询：

```ts
import { surql, Surreal, Thing } from "@tai-kun/surrealdb";

const CreateUserQuery = surql`
  CREATE ONLY user:foo CONTENT { age: 42 };
`.returns<[{ id: Thing<"user">, age: number }]>()

const db = new Surreal();
await db.connect("ws://localhost:8000");

const results = await db.query(CreateUserQuery);
   // ^? const results: [{ id: Thing<"user">, age: number }]

await db.disconnect();
```

### `.queryRaw()` <Badge variant="instance" /><Badge variant="method" />

执行自定义 SurrealQL。

```ts
queryRaw(
  surql: string | object,
  vars?: { [p: string]: unknown },
  options?: ClientRpcOptions,
): Promise<QueryResult[]>;
```

#### 参数

##### `surql`

SurrealQL 字符串或预先准备好的查询。

##### `vars`

在执行 SurrealQL 时传递的变量。变量名不要以 `$` 开头。

##### `options`

PRC 请求的选项。

#### 返回值

返回一个以 SurrealQL 执行结果的数组为解析结果的 `Promise` 对象。

```ts
type QueryResult = {
  status: "OK";
  time: string;
  result: unknown;
} | {
  status: "ERR";
  time: string;
  result: string;
};
```
