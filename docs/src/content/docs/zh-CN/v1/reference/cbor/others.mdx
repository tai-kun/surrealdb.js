## write

### 导入

```ts
import { write } from "@tai-kun/surrealdb/cbor";
```

### 语法

`write` 是一个辅助函数，可以将所有可编码为 CBOR 格式的 JavaScript 值写入 `Writer`。

```ts
write(writer: Writer, value: unknown, options?: WriteOptions): void;
```

#### 参数

##### `writer`

用于将值写入 `Uint8Array` 的 `Writer`。

##### `value`

要编码为 CBOR 格式的 `Uint8Array` 的 JavaScript 值。`value` 可以接受以下 JavaScript 值：

-  原始值
  - `number`
  - `bigint`
  - `string`
  - `boolean`
  - `null`
  - `undefined`
- 具有 `.toCBOR()` 方法的对象
- 数组或 `Set`
- 普通对象或 `Map`
- `Uint8Array`
- `Simple` 对象
- 可由可选的 `replacer` 替换的值

如果 `value` 不是原始值，并且 `toCBOR` 属性是函数，则优先调用该函数。

##### `options`

写入选项。

`replacer?: Replacer | Replacer[]`
:   编码器默认将大多数值编码为 CBOR，但可以将不符合要求的特殊值替换为可编码的值。此选项可以通过指定函数或函数数组来实现替换过程。如果函数返回从 `Symbol.for("@tai-kun/surrealdb/cbor/continue")` 获得的 `CONTINUE` 符号，则会调用后续函数。如果无法进行任何替换，则会抛出 `SurrealTypeError`。

`isSafeMapKey?: (key: unknown, map: ReadonlyMap<unknown, unknown>) => boolean`
:   此选项是一个函数，用于验证 `Map` 对象的键是否安全。该函数接受 `key` 和 `map` 两个参数。`key` 是键值，`map` 是正在验证的 `Map` 对象。如果该函数返回 `false`，则会抛出 `CborUnsafeMapKeyError`。默认情况下，它会将 `"__proto__"` 或 `"constructor"` 判定为不安全的键。请注意，指定此选项将完全覆盖默认验证。如果要在默认验证中添加自己的验证，则应将 `"__proto__"` 或 `"constructor"` 也判定为不安全的键。

`isSafeObjectKey?: (key: unknown, obj: { readonly [p: string]: unknown }) => boolean`
:   此选项是一个函数，用于验证普通对象的键是否安全。该函数接受 `key` 和 `obj` 两个参数。`key` 是键值，`obj` 是正在验证的普通对象。如果该函数返回 `false`，则会抛出 `CborUnsafeMapKeyError`。默认情况下，它会将 `"__proto__"` 或 `"constructor"` 判定为不安全的键。请注意，指定此选项将完全覆盖默认验证。如果要在默认验证中添加自己的验证，则应将 `"__proto__"` 或 `"constructor"` 也判定为不安全的键。

#### 返回值

无。

## writeNumber

### 导入

```ts
import { writeNumber } from "@tai-kun/surrealdb/cbor";
```

### 语法

`writeNumber` 是一个辅助函数，可以将数字写入 `Writer`。

```ts
writeNumber(writer: Writer, value: number): void;
```

#### 参数

##### `writer`

用于将值写入 `Uint8Array` 的 `Writer`。

##### `value`

数值。对于整数，有效值为 -2^53-1 到 2^53-1。在 JavaScript 中，没有方法可以确定 `.0` 是浮点数，因此在这种情况下，它将被视为整数并编码为 CBOR。例如，`-1.0` 是 `-1`，是一个整数。

#### 返回值

无。

## writeHeader

### 导入

```ts

```

### 语法

`writeHeader` 是一个辅助函数，可以将 CBOR 数据项的头部写入 `Writer`。

```ts
writeHeader(writer: Writer, mt: MajorType, length: number | bigint): void;
```

#### 参数

##### `writer`

用于将值写入 `Uint8Array` 的 `Writer`。

##### `mt`

CBOR 主类型。必须是 0 到 7 之间的整数值。

##### `length`

数据本身或数据的字节长度。

#### 返回值

无。
