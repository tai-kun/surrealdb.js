`Simple` 代表一个由 0 到 255 的整数标识的简单值。

### 导入

```ts
import { Simple } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

创建一个新的 `Simple` 对象。

```ts
new Simple(value: number);
```

#### 参数

##### `value`

0 到 255 之间的整数。

#### 返回值

如果通过 `new` 调用，`Simple` 将返回其实例。

### `.value` <Badge variant="instance" /><Badge variant="property" />

一个 0 到 255 之间的整数，用于标识 `Simple` 对象。
