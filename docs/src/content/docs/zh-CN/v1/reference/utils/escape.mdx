## quoteStr

`quoteStr` 用于将字符串用引号括起来。为了方便起见，即使是空字符串也会被用引号括起来。

### 导入

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";
```

### 使用方法

```ts
function quoteStr(str: string): string;
```

用引号（`"` 或 `'`) 括起字符串。

#### 参数

##### `str`

要括起的字符串。

#### 返回值

返回用引号括起来的字符串。使用的引号取决于给定的字符串中是否包含单引号 (`'`)：

- 如果给定的字符串为空
  - 返回 `''`。
- 如果给定的字符串**不**包含单引号
  - 用单引号括起来。
- 如果给定的字符串**包含**单引号
  - 用双引号括起来。
- 如果给定的字符串同时包含单引号和双引号
  - 用双引号括起来。
  - 双引号使用反斜杠 (`\`) 进行转义。

#### 例子

在以下示例中，将**不**包含单引号的字符串用单引号括起来：

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(``));
console.log(quoteStr(`a`));
console.log(quoteStr(`𩸽`));
console.log(quoteStr(`👍🏽`));
console.log(quoteStr(`パ`));
```

输出：

```js
''
'a'
'𩸽'
'👍🏽'
'パ'
```

在以下示例中，将**包含**单引号的字符串用双引号括起来：

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's`));
console.log(quoteStr(`𩸽's`));
console.log(quoteStr(`'👍🏽`));
console.log(quoteStr(`'パ`));
```

输出：

```js
"cat's"
"𩸽's"
"'👍🏽"
"'パ"
```

在以下示例中，将同时包含单引号和双引号的字符串用双引号括起来，并在需要时进行转义：

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's "toy"`));
console.log(quoteStr(`'\\"\\`));
console.log(quoteStr(`𩸽's "feed"`));
console.log(quoteStr(`'👍🏽"`));
console.log(quoteStr(`'パ"`));
```

输出：

```js
"cat's \"toy\""
"'\\\"\\"
"𩸽's \"feed\""
"'👍🏽\""
"'パ\""
```

## escapeKey

`escapeKey` 与 [`quoteStr`](/zh-CN/reference/utils/quote-str/) 类似，但用于对象键。为了方便起见，即使是空字符串也会被用引号括起来。

### 导入

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";
```

### 使用方法

```ts
function escapeKey(key: string): string;
```

#### 参数

##### `key`

对象的键。

#### 返回值

返回转义后的字符串。如果需要，则用双引号括起来：

- 如果给定的字符串为空
  - 返回 `""`。
- 如果给定的字符串仅由字母数字和下划线组成
  - 直接返回。
- 如果给定的字符串包含字母数字和下划线以外的字符
  - 用双引号括起来。
- 如果给定的字符串包含反斜杠 (`\`)
  - 进行转义。
- 如果给定的字符串包含双引号
  - 进行转义。

#### 例子

在以下示例中，因为字符串仅由字母数字和下划线组成，所以直接获得该字符串：

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(`123`));
console.log(escapeKey(`foo_bar`));
```

输出：

```js
123
foo_bar
```

在以下示例中，将字符串用双引号括起来，并在需要时进行转义：

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(``));
console.log(escapeKey(`foo-bar`));
console.log(escapeKey(`'\\"\\`));
console.log(escapeKey(`𩸽's "feed"`));
console.log(escapeKey(`'👍🏽"`));
console.log(escapeKey(`'パ"`));
```

输出：

```js
""
"foo-bar"
"'\\"\"
"𩸽's \"feed\""
"'👍🏽\""
"'パ\""
```

## escapeRid

`escapeRid` 与 [`quoteStr`](/zh-CN/reference/utils/quote-str/) 类似，但用于记录 ID。为了方便起见，即使是空字符串也会被用引号括起来。

### 导入

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";
```

### 使用方法

```ts
function escapeRid(rid: string): string;
```

#### 参数

##### `rid`

记录 ID 的 ID 部分。

#### 返回值

返回转义后的字符串。如果需要，则用尖括号 (`⟨⟩`) 括起来：

- 如果给定的字符串为空
  - 返回 `⟨⟩`。
- 如果给定的字符串不以数字开头，并且仅由字母数字和下划线组成
  - 直接返回。
- 如果给定的字符串包含尖括号 (`⟩`)
  - 进行转义。
- 如果给定的字符串以数字开头，或者包含字母数字和下划线以外的字符
  - 用尖括号 (`⟨⟩`) 括起来。

#### 例子

在以下示例中，因为字符串不以数字开头，并且仅由字母数字和下划线组成，所以直接获得该字符串：

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";

console.log(escapeRid(`foo_123`));
```

输出：

```js
foo_123
```

在以下示例中，将字符串用尖括号 (`⟨⟩`) 括起来，并在需要时进行转义：

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";

console.log(escapeRid(``));
console.log(escapeRid(`123`));
console.log(escapeRid(`foo-bar`));
console.log(escapeRid(`𩸽's ⟨feed⟩`));
console.log(escapeRid(`'👍🏽⟩`));
console.log(escapeRid(`'パ⟩`));
```

输出：

```js
⟨⟩
⟨123⟩
⟨foo-bar⟩
⟨𩸽's ⟨feed\⟩⟩
⟨'👍🏽\⟩⟩
⟨'パ\⟩⟩
```

## escapeIdent

### 导入

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";
```

### 使用方法

```ts
function escapeIdent(ident: string): string;
```

#### 参数

##### `ident`

标识符。

#### 返回值

返回转义后的字符串。如果需要，则用反引号 (<code>`</code>) 括起来：

- 如果给定的字符串为空
  - 返回 <code>``</code>。{/* TODO(tai-kun): " になる問題解決する */}
- 如果给定的字符串不以数字开头，并且仅由字母数字和下划线组成
  - 直接返回。
- 如果给定的字符串包含反引号 (<code>`</code>)
  - 进行转义。
- 如果给定的字符串以数字开头，或者包含字母数字和下划线以外的字符
  - 用反引号 (<code>`</code>) 括起来。

#### 例子

在以下示例中，因为字符串不以数字开头，并且仅由字母数字和下划线组成，所以直接获得该字符串：

输出：


## 专栏

### 字符串长度

在前面的示例中使用的字符串包含一些字符，这些字符在 JavaScript 中的字符串长度和人类感知的字符串长度不同：

| 类型 | 显示 | `.length` | `Uint8Array` |
| --- | --- | --- | --- |
| ASCII | a | 1 | `[ 0x61 ]` |
| 代理对 | 𩸽 | 2 | `[ 0xF0, 0xA9, 0xB8, 0xBD ]` |
| 变体选择器 | 👍🏽 | 4 | `[ 0xF0, 0x9F, 0x91, 0x8D, 0xF0, 0x9F, 0x8F, 0xBD ]` |
| 组合字符 | パ | 2 | `[ 0xE3, 0x83, 0x8F, 0xE3, 0x82, 0x9A ]` |

如果 `.length` 的值与直觉不符，则需要谨慎考虑对字符串的操作。
