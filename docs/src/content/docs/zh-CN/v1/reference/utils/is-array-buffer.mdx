`isArrayBuffer` 函数用于判断传入的值是否为 `ArrayBuffer` 对象。

### 导入

```ts
import { isArrayBuffer } from "@tai-kun/surrealdb/utils";
```

### 使用方法

```ts
function isArrayBuffer(value: unknown): value is ArrayBuffer;
```

#### 参数

##### `value`

期望为 `ArrayBuffer` 对象的值。

#### 返回值

如果 `value` 为 `ArrayBuffer` 对象，则返回 `true`，否则返回 `false`。

### 解释

在某些版本的 Firefox 中，将 `fetch` 的响应体设置为 `ArrayBuffer` 对象时，虽然它具备了 `ArrayBuffer` 的方法，但 `instanceof` 判定可能会失效。因此，我们使用标签名进行判定。这种方法类似于 Lodash 的行为。
