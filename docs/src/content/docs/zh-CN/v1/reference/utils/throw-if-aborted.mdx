`throwIfAborted` 函数用于检查传入的中止信号是否已中止，如果是，则抛出中止原因。它为不支持 `new AbortSignal().throwIfAborted()` 的环境提供替代方案。

## 导入

```ts
import { throwIfAborted } from "@tai-kun/surrealdb/utils";
```

## 使用方法

```ts
function throwIfAborted(signal: AbortSignal | null | undefined): void;
```

#### 参数

`signal`
:   中止信号。

#### 返回值

无。
