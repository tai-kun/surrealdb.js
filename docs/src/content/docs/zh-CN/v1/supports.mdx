## JavaScript

由于此 SDK 使用了 `bigint`，因此需要 ES2020 或更高版本的 JavaScript 语言规范。预期在以下浏览器中运行：

| 浏览器 | 版本 |
|---|---|
| Chrome  | `>=78` |
| Edge    | `>=79` |
| Safari  | `>=14` |
| Firefox | `>=68` |

## 测试

### 服务器

| 环境 | 版本 | 状态 |
|---|---|---|
| SurrealDB | `^1.5.4` | [GitHub](https://github.com/surrealdb/surrealdb) |
| Node.js   | `18.x`,<br/>`20.x`,<br/>`^22.5.1` | [![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/tai-kun/surrealdb.js/nodejs.yml?branch=v1&logo=Node.js&label=Node.js)](https://github.com/tai-kun/surrealdb.js/actions/workflows/nodejs.yml) |
| Deno      | `^1.44.3` | 未测试 |
| Bun       | `^1.1.13` | [![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/tai-kun/surrealdb.js/bun.yml?branch=v1&logo=Bun&label=Bun)](https://github.com/tai-kun/surrealdb.js/actions/workflows/bun.yml) |

- 请注意，Node.js v22 尚未成为 LTS 版本。
- Deno 环境下尚未测试，但可能可以运行。建议使用 v1.44.3 或更高版本。较低版本存在将 `AbortSignal.timeout()` 误判为资源泄漏的 bug。
- 建议 Bun 使用最新版本。
