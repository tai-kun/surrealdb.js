## 版本

这份文档介绍了[`@tai-kun/surrealdb`](https://www.npmjs.com/package/@tai-kun/surrealdb) 的使用方法，并根据主要版本进行了分类。

- v2: [`/v2`](https://tai-kun.github.io/surrealdb.js/v2)（最新版本）
- v1: [`/v1`](https://tai-kun.github.io/surrealdb.js/v1)

## 国际化

这份文档支持以下语言：

- 日语: [`/ja`](https://tai-kun.github.io/surrealdb.js/ja)（原文）
- 中文: [`/zh-CN`](https://tai-kun.github.io/surrealdb.js/zh-CN)
- 韩语: [`/ko`](https://tai-kun.github.io/surrealdb.js/ko)
- 英语: [`/en`](https://tai-kun.github.io/surrealdb.js/en)

除了日语之外的语言版本，均由机器翻译生成。我们已尽力确保翻译内容的准确性，但仍有可能存在误译或不恰当的表达，敬请谅解。

## 重定向

URL 构成和重定向规则如下：

- 根 URL 为 [`https://tai-kun.github.io/surrealdb.js`](https://tai-kun.github.io/surrealdb.js)。
- 访问根目录（`/`）时，将根据浏览器的语言设置重定向到相应的语言页面。默认语言为英语，同时也会重定向到最新版本的文档。
- 访问 `/<language>`（例如：`/ja`）时，将重定向到该语言的最新版本文档。
- 访问 `/<version>`（例如：`/v1`）时，将重定向到指定版本的文档。语言将根据浏览器的设置自动判断。
- 访问 `/<language>/<version>`（例如：`/zh-CN/v1`）时，将直接显示该语言和指定版本的页面，不会进行重定向。
