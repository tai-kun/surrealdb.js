`Simple` 类型代表一个简单的值，用 0 到 255 之间的整数标识。

### 导入

```ts
import { Simple } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

创建新的 `Simple` 对象。

```ts
new Simple(value: number);
```

#### 参数

##### `value`

一个 0 到 255 之间的整数。

#### 返回值

当通过 `new` 调用时，`Simple` 将返回它的实例。

### `.value` <Badge variant="instance" /><Badge variant="property" />

`Simple` 对象的标识，一个 0 到 255 之间的整数。
