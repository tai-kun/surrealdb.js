`Simple` 是一个简单的值，由 0 到 255 之间的整数标识。

### 导入

```ts
import { Simple } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

创建一个新的 `Simple` 对象。

```ts
new Simple(value: number);
```

#### 参数

##### `value`

0 到 255 之间的整数。

#### 返回值

通过 `new` 调用时，`Simple` 会返回其实例。

### `.value` <Badge variant="instance" /><Badge variant="property" />

标识 `Simple` 对象的 0 到 255 之间的整数。
