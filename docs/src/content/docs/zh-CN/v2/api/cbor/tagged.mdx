`Tagged` 是一个自定义值，使用从 0 到 2^64-1 的整数值进行标识。

### 导入

```ts
import { Tagged } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

```ts
new Tagged<T>(tag: number | BigInt, value: T);
```

#### 参数

##### `tag`

用于标识带标签数据项的从 0 到 2^64-1 的整数值。请注意，在 JavaScript 中，使用 `BigInt` 来表示大于 2^53-1 的数字。

##### `value`

带标签数据项的值。

#### 返回值

如果通过 `new` 调用，`Tagged` 将返回其实例。

### `.tag` <Badge variant="instance" /><Badge variant="property" />

`.tag` 是一个属性，它保存用于标识带标签数据项的从 0 到 2^64-1 的整数值。

### `.value` <Badge variant="instance" /><Badge variant="property" />

`.value` 是一个属性，它存储带标签数据项的值。

### `.toCBOR()` <Badge variant="instance" /><Badge variant="method" />

`.toCBOR` 是一个函数，用于返回用于 CBOR 编码的信息。

```ts
toCBOR(): [tag: number | bigint, value: unknown];
```

#### 参数

无。

#### 返回值

返回用于 CBOR 编码的信息。
