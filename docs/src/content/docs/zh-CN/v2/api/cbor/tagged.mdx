`Tagged` 是由 0 到 2^64-1 之间的整数标识的自定义值。

### 导入

```ts
import { Tagged } from "@tai-kun/surrealdb/cbor";
```

### `.constructor()`

```ts
new Tagged<T>(tag: number | BigInt, value: T);
```

#### 参数

##### `tag`

用于标识带标签数据项的 0 到 2^64-1 之间的整数。需要注意的是，在 JavaScript 中，如果使用超过 2^53-1 的数值，则需要使用 BigInt。

##### `value`

带标签数据项的值。

#### 返回值

当通过 `new` 调用时，`Tagged` 将返回其实例。

### `.tag` <Badge variant="instance" /><Badge variant="property" />

`.tag` 是一个属性，包含用于标识带标签数据项的 0 到 2^64-1 之间的整数。

### `.value` <Badge variant="instance" /><Badge variant="property" />

`.value` 是一个属性，用于存储带标签数据项的值。

### `.toCBOR()` <Badge variant="instance" /><Badge variant="method" />

`.toCBOR` 是一个函数，用于返回 CBOR 编码所需的信息。

```ts
toCBOR(): [tag: number | bigint, value: unknown];
```

#### 参数

无。

#### 返回值

返回 CBOR 编码所需的信息。
