`Future` 是一个表示 SurrealDB [Futures 功能](https://surrealdb.com/docs/surrealql/datamodel/futures) 的类。

### 导入

```ts
import { Future } from "@tai-kun/surrealdb/decodeonly-datatypes";
import { Future } from "@tai-kun/surrealdb/encodable-datatypes";
import { Future } from "@tai-kun/surrealdb/standard-datatypes";
```

### `.constructor()`

创建一个新的 `Future` 对象。

```ts
new Future(source: string);
```

#### 参数

##### `source`

future 块中的值或表达式。

#### 返回值

当通过 `new` 调用时，`Future` 会返回它的实例。

### `.block` <Badge variant="instance" /><Badge variant="property" /><Badge variant="decodeonly" />

future 块中的值或表达式。

```ts
.block: number;
```

### `.toString()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="encodable" />

将 `Future` 对象转换为字符串。

```ts
.toString(): string;
```

#### 参数

无。

#### 返回值

返回 future 块中的值或表达式。

#### 示例

```ts
import { Future } from "@tai-kun/surrealdb/encodable-datatypes";

const future = new Future(`time::now()`);
console.log(future.toString());
//-> time::now()
```

### `.toSurql()` <Badge variant="instance" /><Badge variant="method" /><Badge variant="encodable" />

将 `Future` 对象转换为可以嵌入 SurrealQL 的字符串。类似于 `.toString()`，但块用波括号 (`{}`) 括起来，并用 `<future>` 转换。

```ts
.toSurql(): string;
```

#### 参数

无。

#### 返回值

返回可以嵌入 SurrealQL 的 future。

#### 示例

```ts
import { Future } from "@tai-kun/surrealdb/encodable-datatypes";

const future = new Future(`time::now()`);
console.log(future.toSurql());
//-> <future>{time::now()}
```

### `.surql()` <Badge variant="static" /><Badge variant="method" /><Badge variant="standard" />

此方法可以帮助你编写 SurrealQL 代码。你可以使用模板字面量来编写 SurrealQL 代码。嵌入模板字面量的值会自动转换为 SurrealQL。另外，如果你在 VSCode 中安装了 [SurrealDB 官方扩展](https://marketplace.visualstudio.com/items?itemName=surrealdb.surrealql)，则模板字面量的语法高亮会生效。

```ts
.surql(texts: TemplateStringsArray, ...values: unknown[]): string;
```

#### 参数

##### `texts`

构成模板字面量的字符串数组。

##### `values`

要嵌入到 `texts` 中每个元素之间的值。

#### 返回值

SurrealQL 代码。

#### 示例

```ts
import { Future, Thing } from "@tai-kun/surrealdb/standard-datatypes";

const foo = "foo";
const rid = new Thing("person", "tai-kun");
const future = new Future(Future.surql`
  LET $a = ${foo} + ${Future.raw("'-'")};
  LET $b = type::string(${rid});
  string::concat($a, $b)
`);
console.log(future.toSurql());
//-> <future>{
//->   LET $a = 'foo' + '-';
//->   LET $b = type::string(r'person:⟨tai-kun⟩');
//->   string::concat($a, $b)
//-> }
```

如果不用 `.surql()` 的话，上面的例子就需要这样写：

```ts
import { Future, Thing } from "@tai-kun/surrealdb/standard-datatypes";
import { toSurql } from "@tai-kun/surrealdb/utils";

const foo = "foo";
const rid = new Thing("person", "tai-kun");
const future = new Future(/*surql*/ `
  LET $a = ${toSurql(foo)} + ${"'-'"};
  LET $b = type::string(${rid.toSurql()});
  string::concat($a, $b)
`);
console.log(future.toSurql());
//-> <future>{
//->   LET $a = 'foo' + '-';
//->   LET $b = type::string(r'person:⟨tai-kun⟩');
//->   string::concat($a, $b)
//-> }
```

- 你需要使用 `toSurql` 辅助函数或 `.toSurql()` 方法将字符串或对象转换为 SurrealQL 代码。
- 你可以使用 `Future.raw` 来嵌入不需要转换为 SurrealQL 代码的字符串。

### 发展

#### `Future` 对象的通用判断

建议使用 `isFuture` 函数来判断一个值是否是 `Future` 对象，而不要依赖预设。该函数不使用 `instanceof`，因此在验证值的预设未知时很有用。

```ts
import { isFuture } from "@tai-kun/surrealdb";
```
