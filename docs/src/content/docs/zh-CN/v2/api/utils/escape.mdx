## quoteStr

`quoteStr` 函数用于将字符串用引号包围。为了方便，空字符串也会用引号包围。

### 导入

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";
```

### 用法

```ts
function quoteStr(str: string): string;
```

用引号 (`"` 或 `'`) 包围字符串。

#### 参数

##### `str`

要包围的字符串。

#### 返回值

返回用引号包围的字符串。包围使用的引号取决于给定字符串是否包含单引号 (`'`)：

- 如果给定字符串为空
  - 返回 `''`。
- 如果给定字符串不包含单引号
  - 用单引号包围。
- 如果给定字符串包含单引号
  - 用双引号包围。
- 如果给定字符串同时包含单引号和双引号
  - 用双引号包围。
  - 用反斜杠 (`\`) 对双引号进行转义。

#### 例子

以下示例将用单引号包围**不包含**单引号的字符串：

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(``));
console.log(quoteStr(`a`));
console.log(quoteStr(`𩸽`));
console.log(quoteStr(`👍🏽`));
console.log(quoteStr(`パ`));
```

输出：

```js
''
'a'
'𩸽'
'👍🏽'
'パ'
```

以下示例将用双引号包围**包含**单引号的字符串：

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's`));
console.log(quoteStr(`𩸽's`));
console.log(quoteStr(`'👍🏽`));
console.log(quoteStr(`'パ`));
```

输出：

```js
"cat's"
"𩸽's"
"'👍🏽"
"'パ"
```

以下示例将用双引号包围包含单引号和双引号的字符串，并在需要时进行转义：

```ts
import { quoteStr } from "@tai-kun/surrealdb/utils";

console.log(quoteStr(`cat's "toy"`));
console.log(quoteStr(`'\\"\\`));
console.log(quoteStr(`𩸽's "feed"`));
console.log(quoteStr(`'👍🏽"`));
console.log(quoteStr(`'パ"`));
```

输出：

```js
"cat's \"toy\""
"'\\\"\\"
"𩸽's \"feed\""
"'👍🏽\""
"'パ\""
```

## escapeKey

`escapeKey` 与 [`quoteStr`](/v2/api/utils/quote-str/) 类似，但用于对象键。为了方便，空字符串也会用引号包围。

### 导入

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";
```

### 用法

```ts
function escapeKey(key: string): string;
```

#### 参数

##### `key`

对象的键。

#### 返回值

返回转义后的字符串。如果需要，用双引号包围：

- 如果给定字符串为空
  - 返回 `""`。
- 如果给定字符串不以数字开头，并且仅包含字母数字和下划线
  - 直接返回。
- 如果给定字符串包含双引号
  - 对其进行转义。
- 如果给定字符串包含反斜杠 (`\`)
  - 对其进行转义。
- 如果给定字符串以数字开头，或者包含非字母数字和下划线字符
  - 用双引号包围。

#### 例子

以下示例中，由于字符串仅包含字母数字和下划线，因此得到原始字符串：

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(`foo_bar`));
```

输出：

```js
foo_bar
```

以下示例将用双引号包围字符串，并在需要时进行转义：

```ts
import { escapeKey } from "@tai-kun/surrealdb/utils";

console.log(escapeKey(``));
console.log(escapeKey(`123`));
console.log(escapeKey(`foo-bar`));
console.log(escapeKey(`'\\"\\`));
console.log(escapeKey(`𩸽's "feed"`));
console.log(escapeKey(`'👍🏽"`));
console.log(escapeKey(`'パ"`));
```

输出：

```js
""
"123"
"foo-bar"
"'\\"\"
"𩸽's \"feed\""
"'👍🏽\""
"'パ\""
```

## escapeRid

`escapeRid` 与 [`quoteStr`](/v2/api/utils/quote-str/) 类似，但用于记录 ID。为了方便，空字符串也会用引号包围。

### 导入

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";
```

### 用法

```ts
function escapeRid(rid: string): string;
```

#### 参数

##### `rid`

记录 ID 的 ID 部分。

#### 返回值

返回转义后的字符串。如果需要，用尖括号 (`⟨⟩`) 包围：

- 如果给定字符串为空
  - 返回 `⟨⟩`。
- 如果给定字符串不以数字开头，并且仅包含字母数字和下划线
  - 直接返回。
- 如果给定字符串包含尖括号 (`⟩`)
  - 对其进行转义。
- 如果给定字符串全部由数字组成，或者包含非字母数字和下划线字符
  - 用尖括号 (`⟨⟩`) 包围。

#### 例子

以下示例中，由于字符串不以数字开头，并且仅包含字母数字和下划线，因此得到原始字符串：

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";

console.log(escapeRid(`foo_123`));
```

输出：

```js
foo_123
```

以下示例将用尖括号 (`⟨⟩`) 包围字符串，并在需要时进行转义：

```ts
import { escapeRid } from "@tai-kun/surrealdb/utils";

console.log(escapeRid(``));
console.log(escapeRid(`123`));
console.log(escapeRid(`foo-bar`));
console.log(escapeRid(`𩸽's ⟨feed⟩`));
console.log(escapeRid(`'👍🏽⟩`));
console.log(escapeRid(`'パ⟩`));
```

输出：

```js
⟨⟩
⟨123⟩
⟨foo-bar⟩
⟨𩸽's ⟨feed\⟩⟩
⟨'👍🏽\⟩⟩
⟨'パ\⟩⟩
```

## escapeIdent

### 导入

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";
```

### 用法

```ts
function escapeIdent(ident: string): string;
```

#### 参数

##### `ident`

标识符。

#### 返回值

返回转义后的字符串。如果需要，用反引号 (<code>`</code>) 包围：

- 如果给定字符串为空
  - 返回 <code>``</code>。{/* TODO(tai-kun): " になる問題解決する */}
- 如果给定字符串不以数字开头，并且仅包含字母数字和下划线
  - 直接返回。
- 如果给定字符串包含反引号 (<code>`</code>)
  - 对其进行转义。
- 如果给定字符串以数字开头，或者包含非字母数字和下划线字符
  - 用反引号 (<code>`</code>) 包围。

#### 例子

以下示例中，由于字符串不以数字开头，并且仅包含字母数字和下划线，因此得到原始字符串：

```ts
import { escapeIdent } from "@tai-kun/surrealdb/utils";

console.log(escapeIdent(`foo_123`));
```

输出：

```js

```

以下示例将用反引号 (<code>`</code>) 包围字符串，并在需要时进行转义：

```ts

console.log(escapeIdent(""));
```

## 补充说明

### 字符串长度

前面的示例中使用的字符串包含了在 JavaScript 中的字符串长度与人类感知的字符串长度不同的字符：

| 类型 | 显示 | `.length` | `Uint8Array` |
| --- | --- | --- | --- |
| ASCII | a | 1 | `[ 0x61 ]` |
| 代理对 | 𩸽 | 2 | `[ 0xF0, 0xA9, 0xB8, 0xBD ]` |
| 变体选择器 | 👍🏽 | 4 | `[ 0xF0, 0x9F, 0x91, 0x8D, 0xF0, 0x9F, 0x8F, 0xBD ]` |
| 组合字符 | パ | 2 | `[ 0xE3, 0x83, 0x8F, 0xE3, 0x82, 0x9A ]` |

如果 `.length` 的值与直觉不符，则需要谨慎考虑对字符串的操作。
