`isBrowser` 函数用于判断当前运行的 JavaScript 环境是否为浏览器环境。

### 导入

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";
```

### 使用方法

```ts
function isBrowser(): boolean;
```

#### 参数

无。

#### 返回值

如果当前运行环境为浏览器环境，则返回 `true`；否则返回 `false`。

#### 示例

以下示例演示了如何判断当前运行环境是否为浏览器环境：

```ts
import { isBrowser } from "@tai-kun/surrealdb/utils";

if (isBrowser()) {
  document;
}
```

### 解释

如果全局变量 `document` 不为 `undefined`，则判断为浏览器环境。例如，Deno 将 `window` 对象添加到全局环境，因此如果通过 `window` 对象的存在来判断，则服务器运行时也会被误判为浏览器环境。Web 框架 Remix 将此称为 [Document Guard](https://remix.run/docs/en/main/guides/constraints#document-guard)。
