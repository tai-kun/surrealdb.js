`throwIfAborted` 是一个函数，它会在传入的 AbortSignal 被中止时抛出中止的原因。它为不支持 `new AbortSignal().throwIfAborted()` 的环境提供支持。

## 导入

```ts
import { throwIfAborted } from "@tai-kun/surrealdb/utils";
```

## 使用方法

```ts
function throwIfAborted(signal: AbortSignal | null | undefined): void;
```

#### 参数

`signal`
:   中止信号。

#### 返回值

无。
