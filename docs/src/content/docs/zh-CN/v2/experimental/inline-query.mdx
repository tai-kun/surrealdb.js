内联查询是一个函数，它无需创建 `Surreal` 类的实例即可直接执行一次性自定义查询。其用法与 [`.query()`](/v2/guides/querying#query) 方法非常相似。但需要注意的是，默认情况下使用的是 `JsonFormatter`。

### 导入

```ts
import { query } from "@tai-kun/surrealdb";
```

### 示例

```ts
import { query, rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});

const results = await query("http://127.0.0.1:8000", "RETURN 42", {}, {
  token,
  namespace: "test",
  database: "test",
});
```
