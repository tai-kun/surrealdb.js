内联 RPC 函数无需创建 `Surreal` 类实例即可执行一次性 RPC。使用方法与 `.rpc()` 方法几乎相同。但请注意，默认情况下使用的是 `JsonFormatter`。

### 导入

```ts
import { rpc } from "@tai-kun/surrealdb";
```

### 示例

```ts
import { rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});
```
