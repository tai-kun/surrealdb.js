内联 RPC 是一个函数，可以在不创建 `Surreal` 类实例的情况下，执行一次性的 RPC 操作。其用法与 `.rpc()` 方法几乎相同。但需要注意的是，默认情况下会使用 `JsonFormatter`。

### 导入

```ts
import { rpc } from "@tai-kun/surrealdb";
```

### 示例

```ts
import { rpc } from "@tai-kun/surrealdb";

const token = await rpc("http://127.0.0.1:8000", "signin", {
  params: [{
    user: "root",
    pass: "root",
  }],
});
```
