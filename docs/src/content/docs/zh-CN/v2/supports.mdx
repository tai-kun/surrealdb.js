## JavaScript

该 SDK 使用了 `bigint`，因此需要 JavaScript 语言规范为 ES2020 或更高版本。此外，考虑到该 SDK 在运行时所需的特性，我们预计它可以在以下运行时环境中运行：

| 运行时 | 版本 |
| --- | --- |
| Node.js | `18.x`,`20.x`,`^22.5.1` |
| Deno    | `1.x`, `2.x` |
| Bun     | `1.x` |
| Chrome  | `>=78` |
| Edge    | `>=79` |
| Safari  | `>=14` |
| Firefox | `>=68` |

## 测试

我们使用 Vitest 和 Playwright 进行测试。由于我们在支持 Vitest 和 Playwright 的最低版本运行时环境中进行测试，因此实际上该 SDK 可能在比以下列出的版本更旧的运行时环境中也能运行。
