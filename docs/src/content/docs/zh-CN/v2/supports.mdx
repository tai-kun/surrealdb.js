## JavaScript

该 SDK 使用 `bigint`，因此需要 ES2020 或更高版本的 JavaScript 语言规范。此外，考虑到 SDK 在运行时所需的特性，预计它可以在以下运行时环境中运行：

| 运行时 | 版本 |
| --- | --- |
| Node.js | `18.x`,`20.x`,`^22.5.1` |
| Deno | `^1.44.3` |
| Bun | `^1.1.13` |
| Chrome  | `>=78` |
| Edge    | `>=79` |
| Safari  | `>=14` |
| Firefox | `>=68` |

## 测试

测试使用 Vitest 和 Playwright。由于我们在测试时使用的是这些工具可以工作的最低版本运行时，因此实际上可能在比下面列出的版本更旧的运行时中也能工作。

### 服务器

| 运行时 | 版本 | 状态 |
